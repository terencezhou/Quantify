# 缩量主升浪 + 趋势持股 — 完整交易策略

## 一、策略核心理念

### 1.1 交易什么

做「缩量蓄力 → 放量主升」这一段确定性最高的趋势行情。

一只股票的完整上涨周期通常经历四个阶段：

```
底部震荡 → 缩量整理（蓄力）→ 放量主升 → 加速赶顶
```

大多数散户在底部震荡时不敢买、主升浪中段才追进、加速赶顶时贪心不走。本策略的核心是：**在蓄力阶段发现它，在主升启动时买入，在趋势破位时离场**——只吃中间最确定的一段。

### 1.2 预期差视角

所有短线策略都在交易「预期差」（参见 insight.md）。本策略的预期差是：

> **市场认为横盘缩量 = 无聊 / 弱势，实际上 = 主力吸筹完毕、即将启动。**

当一只股票缩量横盘 20+ 天、散户失去兴趣纷纷离场时，恰恰说明浮筹已被清洗、筹码集中度提升。一旦放量突破，就是主升浪的起点。

### 1.3 适用前提

| 条件 | 要求 |
|------|------|
| 时间投入 | 每天 10-30 分钟（收盘后复盘） |
| 盯盘要求 | 不需要盯盘，收盘后看 K 线做筛选 |
| 持仓周期 | 5-20 个交易日 |
| 心态要求 | 能接受横盘等待、能拿住趋势、能执行止损 |
| 不适合 | 熊市单边下跌期、追求日内暴利 |

## 二、选股条件

### 2.1 五大核心条件

同时满足以下五个条件的股票进入「观察池」：

#### 条件 1：缩量

近期成交量相对于历史成交量显著萎缩，说明卖方力量衰竭、浮筹被清洗。

```
volume_ratio = MA(成交量, 10) / MA(成交量, 60)

入选条件: volume_ratio < 0.6

含义:
  < 0.4  — 极度缩量，蓄力充分
  0.4~0.6 — 明显缩量，可关注
  > 0.6  — 缩量不充分，不入选
```

#### 条件 2：横盘整理

股价波动率相对于自身历史显著收敛，形成窄幅震荡区间。

```
σ_recent  = std(近 15 日收益率)
σ_history = std(前 60 日收益率)
VR = σ_recent / σ_history

入选条件: VR < 0.5

含义:
  VR < 0.3  — 极度收敛（非常安静的横盘）
  0.3~0.5  — 明显收敛，可关注
  > 0.5    — 波动仍大，不算横盘
```

注意：使用相对波动率而非绝对振幅，是因为不同股票的正常波动水平差异很大。银行股日常波动 1%，科技股日常波动 3%，用同一个绝对阈值无法兼顾。

#### 条件 3：20 日均线向上

MA20 持续上行，说明中期趋势向上，横盘是"强势整理"而非"弱势盘跌"。

```
slope_ma20 = (MA20_today - MA20_5days_ago) / MA20_5days_ago

入选条件: slope_ma20 > 0 且连续 >= 5 个交易日

排除: MA20 走平或向下的横盘（可能是下跌中继）
```

#### 条件 4：重心上移

虽然横盘震荡，但每次回踩的低点在逐步抬高，说明有资金在低位持续承接。

```
low_avg_recent = mean(近 5 日最低价)
low_avg_prev   = mean(前 5~10 日最低价)

入选条件: low_avg_recent > low_avg_prev

更严格版本: 近 20 日内，以 5 日为窗口滑动，
           每个窗口的最低价均值 >= 前一个窗口的最低价均值
```

#### 条件 5：换手率适中

排除两种极端：换手率过低的僵尸股/庄股，和换手率过高的投机股。

```
avg_turnover = MA(换手率, 10)

入选条件: 1% <= avg_turnover <= 5%

排除:
  < 1%  — 流动性差，主力可能高度控盘（庄股），不适合趋势交易
  > 5%  — 筹码交换过于频繁，说明分歧大，不是蓄力状态
```

### 2.2 补充过滤条件

| 条件 | 规则 | 原因 |
|------|------|------|
| 流通市值 | 30 亿 ~ 500 亿 | 小盘股暴涨暴跌难以把控，大盘股弹性不足 |
| 涨幅位置 | 距 120 日最低价涨幅 < 50% | 已经涨太多的不追，避免在山腰接盘 |
| 排除 ST | 名称不含 ST / *ST | 退市风险 |
| 排除北交所 | 代码不以 8/4 开头 | 流动性和涨跌幅机制不同 |
| 排除次新股 | 上市时间 > 60 个交易日 | 次新波动大、无历史支撑参考 |

### 2.3 选股条件总结

```
观察池 = {
  全市场股票
  ∩ 缩量 (volume_ratio < 0.6)
  ∩ 横盘 (VR < 0.5)
  ∩ MA20 向上 (slope > 0, 持续 >= 5天)
  ∩ 重心上移 (low_avg_recent > low_avg_prev)
  ∩ 换手率适中 (1% ~ 5%)
  ∩ 流通市值 30~500亿
  ∩ 涨幅位置 < 50%
  ∩ 非 ST / 非北交所 / 非次新
}
```

## 三、买入信号

选股条件筛出的是「有蓄力迹象」的股票，但蓄力不等于一定会启动。需要等待明确的启动信号再买入。

### 3.1 两阶段确认

```
阶段 1: 进入观察池   — 满足五大选股条件（蓄力中）
阶段 2: 触发买入信号 — 放量突破 + 均线发散 + K 线确认（主升启动）
```

### 3.2 买入确认信号（三个同时满足）

#### 信号 A：放量突破

```
当日成交量 > MA(成交量, 5) × 1.5
且
当日收盘价 > max(近 N 日收盘价)    // N = 横盘天数

含义: 沉寂已久的成交量突然放大，同时价格创出近期新高，
     说明有新资金入场推动突破。
```

为什么用 1.5 倍而非 2 倍：缩量横盘后的正常突破量通常在 1.5-2.5 倍之间。用 2 倍会漏掉一些主力控盘良好、不需要太大量就能突破的优质标的。

#### 信号 B：均线发散

```
MA5 > MA10 > MA20
且
(MA5 - MA20) / MA20 > 前一日的 (MA5 - MA20) / MA20

含义: 短期均线在中期均线上方，且距离在扩大，
     说明上涨动能在加速而非昙花一现。
```

#### 信号 C：K 线形态确认

```
当日收阳线 (收盘 > 开盘)
且
实体占比 = (收盘 - 开盘) / (最高 - 最低) > 0.6

含义: 实体饱满的阳线，说明多头强势推进，
     排除长上影线（盘中冲高回落的假突破）。
```

### 3.3 环境过滤

即使个股信号完美，也需要大环境配合（对接现有系统模块）：

| 环境条件 | 数据来源 | 规则 |
|----------|---------|------|
| 市场非冰点 | `market_temperature` | score >= 30 |
| 概念/行业有热度 | `concept_hot` / `industry_temperature` | 所属板块处于启动或发酵期 |
| 市场情绪方向 | `market_temperature` | 今日 score >= 昨日 score（情绪不在退化） |

环境过滤不是硬性否决，而是分级：
- 环境全满足 → 正常仓位买入
- 市场温度 30-40 → 半仓买入
- 市场温度 < 30 → 不买入，等待

### 3.4 买入时机

```
信号出现日: 收盘后确认 A + B + C 三条件满足
执行买入:   次日集合竞价或开盘后 15 分钟内

挂单策略:
  - 如果信号日涨停（无法确认次日是否延续），次日开盘观察 15 分钟再决定
  - 如果信号日涨幅 3%~7%，次日以信号日收盘价附近挂单
  - 如果次日低开 > 2%，观望不追（可能是假突破）
```

## 四、仓位管理

### 4.1 基本原则

| 规则 | 参数 | 原因 |
|------|------|------|
| 单只上限 | 总资金的 20% | 单票风险可控（最大亏损 = 20% × 8% = 1.6%） |
| 总持仓上限 | 3-5 只 | 集中精力跟踪，但不押注单一标的 |
| 同板块上限 | 2 只 | 避免板块系统性风险 |

### 4.2 分批建仓

```
首次建仓: 计划仓位的 60%
         触发条件: 买入确认信号出现

加仓:     计划仓位的 40%
         触发条件: 买入后 3 个交易日内，股价站稳突破位上方
                  且成交量维持在 MA5 以上
                  且 MA5 > MA10 > MA20 仍然成立

不加仓:   如果 3 日内回落至突破位以下 → 不加仓，按止损规则处理
```

### 4.3 市场温度仓位调节

根据 `market_temperature` 动态调整总仓位上限：

| 市场温度 | 总仓位上限 | 说明 |
|----------|-----------|------|
| >= 50 | 100% | 主升/发酵期，正常操作 |
| 40-50 | 80% | 混沌期，略微谨慎 |
| 30-40 | 50% | 偏弱，控制风险 |
| 25-30 | 20% | 接近冰点，仅保留最强标的 |
| < 25 | 0% | 冰点期，空仓等待 |

## 五、止损规则

止损是策略的生命线。三条规则任一触发即执行，无条件、不犹豫。

### 5.1 硬止损（优先级最高）

```
触发条件: 收盘价 < 买入均价 × (1 - 8%)
执行:     次日开盘卖出，不等反弹

示例: 买入均价 20 元 → 止损线 18.4 元
     一旦收盘跌破 18.4，次日无条件清仓
```

为什么是 8%：
- 本策略买入点是放量突破横盘区间，如果突破有效，正常回踩不应跌破 8%
- 跌破 8% 大概率说明突破失败、主力未能有效拉升
- 从仓位角度：单票 20% 仓位 × 8% 亏损 = 总资金亏损 1.6%，完全可承受

### 5.2 趋势止损

```
触发条件: 收盘价连续 2 个交易日 < MA20
执行:     第 3 日开盘卖出

注意:
  - 必须是"收盘价"跌破，不是盘中瞬间跌破
  - 必须"连续 2 日"，排除单日洗盘的假信号
  - MA20 是本策略的趋势生命线，跌破即认为趋势结束
```

### 5.3 板块止损

```
触发条件: 所属概念/行业进入退潮期（concept_hot / industry_temperature 判定）
         且 个股无独立走势（近 3 日涨跌幅 < 板块涨跌幅）
执行:     减仓至 1/2，观察 2 日，若仍无起色则清仓

注意:
  如果板块退潮但个股逆势走强（独立行情），可继续持有
  独立行情判定: 近 3 日个股涨幅 > 板块涨幅 + 3%
```

### 5.4 止损优先级

```
硬止损 (8%) > 趋势止损 (破MA20) > 板块止损
```

硬止损触发时不看其他条件，直接执行。

## 六、止盈 / 退出规则

### 6.1 移动止盈

趋势行情中，不预设固定目标价。而是跟踪最高点，当从最高点回撤达到阈值时止盈。阈值随盈利幅度递增——赚得越多，给的回撤空间越大：

| 当前盈利幅度 | 回撤止盈阈值 | 含义 |
|-------------|-------------|------|
| < 10% | 5% | 刚启动，回撤空间小，保护利润 |
| 10% - 20% | 8% | 趋势确认，适度放宽 |
| 20% - 30% | 10% | 主升段，给更多空间 |
| > 30% | 12% | 大涨后允许更多波动 |

```
计算示例:
  买入价 20 元，当前最高曾到 25 元（盈利 25%）
  回撤阈值 = 10%（对应 20%-30% 盈利档）
  止盈触发价 = 25 × (1 - 10%) = 22.5 元

  如果收盘跌到 22.5 元 → 次日开盘止盈
  如果后续继续涨到 28 元 → 最高点更新为 28
  新止盈触发价 = 28 × (1 - 12%) = 24.64 元（盈利 > 30% 档）
```

### 6.2 趋势破位退出

```
触发条件: MA5 下穿 MA20 且 MA20 斜率由正转负（拐头向下）
执行:     次日开盘全部卖出

与趋势止损的区别:
  - 趋势止损: 股价跌破 MA20（看价格位置）
  - 趋势破位: MA5 死叉 MA20 且 MA20 拐头（看均线结构）
  - 趋势破位通常晚于趋势止损触发，是兜底退出条件
```

### 6.3 时间止损

```
触发条件: 买入后满 10 个交易日，盈利不足 5%
执行:     减仓至 1/2

触发条件: 买入后满 15 个交易日，盈利不足 5%
执行:     全部清仓

原因: 如果突破后 10 天内没有展开主升浪，
     大概率是假突破或主力尚未准备好，
     资金不应空耗在没有动能的标的上。
```

### 6.4 退出优先级

```
硬止损 > 趋势止损 > 移动止盈 > 趋势破位 > 时间止损 > 板块止损
```

## 七、每日操作流程

整个策略设计为每天只需 10-30 分钟，在 15:00 收盘后执行：

```
15:30  Step 1 — 市场环境 (3 分钟)
       ├─ 运行系统，查看 market_temperature 分数与阶段
       ├─ 确认当前仓位上限（参照市场温度仓位表）
       └─ 扫一眼概念热度 Top 5（有无新启动的板块）

       Step 2 — 选股池 (5 分钟)
       ├─ 查看系统输出的观察池列表
       ├─ 新进入池中的标的: 标记关注
       ├─ 已在池中的标的: 检查是否触发买入信号 (A+B+C)
       └─ 触发买入的: 记录计划买入价和仓位

       Step 3 — 持仓检查 (3 分钟)
       ├─ 逐只检查: 硬止损 / 趋势止损 / 移动止盈 / 时间止损
       ├─ 触发退出的: 记录次日卖出计划
       └─ 持仓正常的: 确认趋势是否完好（MA5 > MA20?）

       Step 4 — 次日计划 (2 分钟)
       ├─ 整理买入挂单（价格、数量）
       ├─ 整理卖出挂单（价格、数量）
       └─ 次日 9:15 前完成挂单

21:00  可选: 如果有新买入计划，快速查看该股基本面
       ├─ 有无重大利空 / 业绩预警
       └─ 有无解禁 / 减持公告
```

## 八、量化评分模型

为了对观察池中的标的排优先级，设计一个 0-100 的综合评分：

### 8.1 评分维度

| 维度 | 权重 | 评分范围 | 说明 |
|------|------|---------|------|
| A. 缩量质量 | 20% | 0-100 | volume_ratio 越低越好 |
| B. 横盘质量 | 20% | 0-100 | VR 越低、持续天数越长越好 |
| C. 趋势强度 | 20% | 0-100 | MA20 斜率、重心上移幅度 |
| D. 量价配合 | 15% | 0-100 | 涨时放量跌时缩量（vp_ratio） |
| E. 突破强度 | 15% | 0-100 | 放量倍数、突破幅度、K 线实体 |
| F. 环境加分 | 10% | 0-100 | 市场温度 + 板块热度 |

### 8.2 各维度评分

#### A. 缩量质量 (20%)

```
base = 0
if volume_ratio < 0.3:   base = 90
elif volume_ratio < 0.4:  base = 75
elif volume_ratio < 0.5:  base = 55
elif volume_ratio < 0.6:  base = 35
else:                     base = 0

连续缩量天数加分: +3/天 (最多 +15)
换手率 < 2%: +10（筹码锁定好）
换手率 > 4%: -10（分歧仍在）
```

#### B. 横盘质量 (20%)

```
base = clip((0.5 - VR) / 0.4, 0, 1) × 70    // VR 越低分越高

横盘持续天数:
  15-20 天: +10
  20-30 天: +20
  30-45 天: +25
  > 45 天:  +30

重心上移幅度（low_avg 提升比例）:
  > 3%:  +10
  1%-3%: +5
  < 1%:  +0
```

#### C. 趋势强度 (20%)

```
MA20 斜率:
  slope > 0.5%:  70
  slope 0.3-0.5%: 50
  slope 0.1-0.3%: 30
  slope <= 0.1%:  10

MA20 向上持续天数:
  >= 20 天: +20
  10-20 天: +15
  5-10 天:  +10

MA5 > MA10 > MA20 (多头排列): +10
```

#### D. 量价配合 (15%)

```
复用 daily_k.md 的 vp_ratio 思路:
  vp_ratio = 上涨日均量 / 下跌日均量 (近 20 日)

  vp_ratio > 1.5:  90（涨时放量跌时缩量，主力吸筹特征明显）
  vp_ratio 1.2-1.5: 65
  vp_ratio 1.0-1.2: 40
  vp_ratio 0.8-1.0: 20
  vp_ratio < 0.8:   0 （跌时放量涨时缩量，出货特征）
```

#### E. 突破强度 (15%)

仅在触发买入信号时计算，观察池阶段此项为 0：

```
放量倍数:
  vol_ratio = 当日量 / MA5量
  > 3.0:  40
  2.0-3.0: 30
  1.5-2.0: 20
  < 1.5:   0（不算放量）

突破幅度:
  breakout = (收盘 - 近N日最高) / 近N日最高
  > 5%:    30
  3%-5%:   20
  1%-3%:   10
  < 1%:    0

K 线实体占比:
  > 80%:  30
  60%-80%: 20
  < 60%:  10
```

#### F. 环境加分 (10%)

```
市场温度 (50%):
  直接映射: score / 100 × 50

板块热度 (50%):
  所属概念/行业处于启动或发酵期: 80
  处于平稳期:                  50
  处于高潮或退潮期:            20
  无明显板块归属:              40
```

### 8.3 总分与操作

```
总分 = A×0.20 + B×0.20 + C×0.20 + D×0.15 + E×0.15 + F×0.10
```

| 总分 | 级别 | 操作 |
|------|------|------|
| >= 75 | A — 极佳 | 优先买入，可用满仓位（单票 20%） |
| 60-74 | B — 良好 | 正常买入，计划仓位的 70% |
| 45-59 | C — 一般 | 仅观察，不操作 |
| < 45 | D — 较弱 | 移出观察池 |

### 8.4 硬性否决

无论评分多高，出现以下情况直接排除：

1. 近 5 日有跌停（可能是暴雷，不是正常回调）
2. 市场处于冰点期（market_temperature < 25）
3. 该股近 20 日有重大利空公告（减持/业绩预警/立案调查）
4. 流通市值 < 30 亿或 > 500 亿

## 十、风险收益预期

基于策略逻辑和 A 股趋势行情的统计特征，给出理性预期（非回测结论）：

| 指标 | 预期值 | 依据 |
|------|--------|------|
| 胜率 | 55%-65% | 缩量横盘后放量突破的成功率在 A 股中较高 |
| 盈亏比 | 2:1 ~ 3:1 | 单笔盈利目标 10%-30%，止损 8% |
| 持仓周期 | 5-20 个交易日 | 主升浪通常持续 1-4 周 |
| 最大单笔亏损 | -8% | 硬止损保护 |
| 最大单笔亏损占总资金 | -1.6% | 单票 20% × 8% |
| 年度交易次数 | 30-60 次 | 平均每月 3-5 次 |
| 预期年化收益 | 20%-40% | 保守估计，假设满仓轮动 |

### 收益情景分析

```
假设: 初始资金 50 万，每次交易 10 万（20% 仓位）

保守情景（胜率 55%, 平均盈利 12%, 平均亏损 6%）:
  50 次交易 → 28 胜 22 负
  盈利: 28 × 10万 × 12% = 33.6 万
  亏损: 22 × 10万 × 6%  = 13.2 万
  净利润: 20.4 万 → 年化 40.8%

一般情景（胜率 50%, 平均盈利 15%, 平均亏损 7%）:
  50 次交易 → 25 胜 25 负
  盈利: 25 × 10万 × 15% = 37.5 万
  亏损: 25 × 10万 × 7%  = 17.5 万
  净利润: 20.0 万 → 年化 40.0%

悲观情景（胜率 45%, 平均盈利 10%, 平均亏损 7%）:
  50 次交易 → 23 胜 27 负
  盈利: 23 × 10万 × 10% = 23.0 万
  亏损: 27 × 10万 × 7%  = 18.9 万
  净利润: 4.1 万 → 年化 8.2%
```

即使悲观情景下，由于止损控制严格，最差也不会亏损，只是收益较低。

## 十一、策略局限与注意事项

### 11.1 策略固有局限

| 局限 | 说明 | 应对 |
|------|------|------|
| 熊市失效 | 单边下跌时，所有横盘都是下跌中继 | 结合 market_temperature 空仓 |
| 信号稀缺 | 牛市中缩量横盘的股票很少 | 接受空仓等待，不勉强入场 |
| 假突破 | 放量突破后又跌回横盘区间 | 止损保护，不心存幻想 |
| 板块依赖 | 趋势股需要板块资金持续推动 | 板块退潮及时止损 |
| 错过快速轮动 | 持股周期长，可能错过其他热点 | 专注本策略，不频繁切换 |

### 11.2 心态管理

1. **能等**：观察池可能连续数日为空，这是正常的。不要因为"好久没操作"就降低标准
2. **能拿**：趋势行情中，盘中波动是正常的。不要因为盘中回调就恐慌卖出
3. **能砍**：止损触发时果断执行。一次止损亏 1.6%（总资金），远小于死扛可能的 10%+
4. **不贪**：移动止盈触发就走。趋势终会结束，不要试图卖在最高点

### 11.3 与打板策略的关系

本策略和现有的打板/龙头策略完全独立，不互相干扰：

| 维度 | 打板策略 | 趋势策略 |
|------|---------|---------|
| 选股来源 | 涨停池 / 龙虎榜 | 全市场技术筛选 |
| 持仓周期 | 1-3 天 | 5-20 天 |
| 盯盘要求 | 高（需实时） | 低（收盘后） |
| 仓位 | 可独立分配 | 可独立分配 |
| 报告命令 | `buy` | `trend`（待实现） |

两者可以共存：用总资金的一部分做趋势持股（稳健），另一部分做打板（进攻）。

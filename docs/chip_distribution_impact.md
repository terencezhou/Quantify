# 筹码分布数据对现有策略的优化分析

## 数据来源

东财接口 `stock_cyq_em`，每只股票返回最近约 90 个交易日的筹码分布数据，缓存在 `stock_data/chips/` 目录下。

### 字段说明

| 列名 | 含义 | 示例（000001 平安银行 2026-03-05） |
|------|------|-----------------------------------|
| `日期` | 交易日 | 2026-03-05 |
| `获利比例` | 当日收盘价下获利筹码占比 | 0.0524 (5.24%) |
| `平均成本` | 全部筹码的加权平均持仓成本 | 11.49 |
| `90成本-低` | 90% 筹码分布的下界 | 10.80 |
| `90成本-高` | 90% 筹码分布的上界 | 12.39 |
| `90集中度` | (高-低)/(高+低)，越小筹码越集中 | 0.0689 (6.89%) |
| `70成本-低` | 70% 筹码分布的下界 | 10.95 |
| `70成本-高` | 70% 筹码分布的上界 | 12.06 |
| `70集中度` | 70% 筹码区间的集中度 | 0.0482 (4.82%) |

## 策略优化分析

### 1. sell_point — 卖出点分析 ⭐⭐⭐⭐⭐ 最高价值

筹码数据能直接强化其中至少 3 个维度：

**维度四（微观·技术形态）**：当前只靠 K 线和均线判断支撑/压力。加入筹码后：
- `平均成本` 就是天然的压力/支撑分界线——持仓价在平均成本上方，获利筹码多，抛压大；反之抛压小
- `90成本-高` 是上方最大压力位，股价接近这个位置时卖出信号应该加强

**维度五（微观·量价背离）**：当前只看量价关系。筹码能补充"谁在卖"的视角：
- `获利比例` 连日走高（比如从 50% → 90%），说明大量筹码进入获利区间，获利盘兑现压力陡增
- `70集中度` 突然变大（筹码发散），说明市场分歧加大，是主力分批出货的信号

**维度七（盈亏管理）**：当前只用买入价算浮盈。加入筹码后：
- 可以判断持仓成本在全市场筹码中的位置——如果成本高于 `90成本-高`，说明比 90% 的人买得贵，被动性极强

---

### 2. shrink_volume_support_signal — 缩量触支撑 ⭐⭐⭐⭐⭐ 最高价值

文档已经在"支撑强度"维度里把筹码峰列为第一优先级支撑类型，但目前只是设计，没有对接真实数据。有了筹码缓存后可以直接落地：

- `70成本-低` / `90成本-低` → 直接作为筹码峰支撑位的量化值
- `70集中度` → 集中度越小，支撑越可靠（筹码密集 = 共识强）
- `获利比例` 极低（如 < 10%）+ 缩量 → 最强的"卖无可卖"信号，支撑有效概率最高

这个策略是筹码数据价值兑现最直接的场景。

---

### 3. leader_stock — 龙头股识别 ⭐⭐⭐ 中等价值

当前龙头打分是五维度（连板、先手、封板、辨识、助攻），全是基于涨停/封板等市场行为。筹码能补充两个角度：

**筹码锁定度**：真正的龙头股，主力筹码高度集中且不轻易松动
- `70集中度` 很小（如 < 5%）→ 筹码极度集中，说明有主力控盘
- `获利比例` 高但股价不跌 → 获利盘不出，说明主力锁仓坚定
- 这可以作为龙头"确认维度"的加分项，区分真龙头和一日游

**生命周期判断**：当前靠连板天数和市场情绪判断龙头的生命周期（启动→加速→高潮→退潮）
- `获利比例` 从持续走高突然骤降 → 大量获利筹码被套，可能是高潮转退潮的拐点
- `平均成本` 快速上移 → 高位换手充分，筹码从低位搬到高位，龙头行情接近尾声

---

### 4. industry_temperature — 行业温度 ⭐⭐ 较低价值

行业温度是中观层面，筹码是个股数据，需要聚合才有意义。可以做但优先级低：

- 将板块内所有个股的 `获利比例` 取均值 → "板块整体获利水平"。如果一个热门板块的平均获利比例已经 > 80%，说明板块内大量筹码获利，退潮风险大
- 板块内个股的 `70集中度` 均值 → 如果集中度普遍变大，说明板块内分歧加剧

但这需要对全板块个股都拉取筹码数据，目前 `fetch_chips` 全量拉取很慢（5000 只 × 13 秒），实操成本高。

---

### 5. market_temperature — 市场温度 ⭐ 价值最低

市场温度完全是宏观指标（涨停数、跌停数、大面数、封板率等），筹码数据几乎帮不上忙。硬要做的话：
- 全市场平均获利比例可以作为一个辅助的市场情绪指标（类似"套牢盘指数"）
- 但计算成本太高（需要全市场筹码），且与现有指标高度相关，边际收益低

---

## 优化优先级总结

| 策略 | 筹码优化价值 | 建议优先级 |
|------|-------------|-----------|
| `sell_point` | ⭐⭐⭐⭐⭐ | **第一**，直接增强 3 个维度 |
| `shrink_volume_support_signal` | ⭐⭐⭐⭐⭐ | **第一**（并列），支撑位量化的核心数据源 |
| `leader_stock` | ⭐⭐⭐ | 第二，补充确认维度和生命周期判断 |
| `industry_temperature` | ⭐⭐ | 第三，需要聚合，实操成本高 |
| `market_temperature` | ⭐ | 不建议，性价比太低 |

## 筹码 + 其他信号的增强买入组合

### 一、筹码 + 量价关系（日K数据）

#### 信号 ① 单峰密集 + 放量突破

```
条件: 70集中度 < 4%（筹码极度集中，形成单峰）
     + 股价从下方突破「平均成本」
     + 当日量比 > 2.0

含义: 所有人成本几乎一样，突破成本线意味着全部解套，
     没有套牢抛压，放量确认资金认可这个方向。
     这是筹码分析里最经典的买入信号。
```

#### 信号 ② 获利比例极低 + 地量 + 止跌阳线

```
条件: 获利比例 < 5%（几乎所有人被套）
     + 成交量 < MA5 的 50%（地量）
     + 当日收阳线

含义: 比「缩量触支撑」更极端的版本。
     95%的人被套却没人卖 = 卖无可卖，
     地量阳线说明有新资金开始试探性买入。
```

#### 信号 ③ 平均成本粘合 + 股价起飞

```
条件: 连续 5 日「平均成本」变化 < 0.5%（成本几乎不动）
     + 突然单日涨幅 > 5%

含义: 平均成本长期不动说明筹码完全锁死，没有换手，
     突然拉升 = 主力完成吸筹后启动。
     成本不动的天数越长，爆发力越强。
```

---

### 二、筹码 + 资金流（fund_flow 数据）

#### 信号 ④ 底部集中 + 主力持续净流入

```
条件: 70集中度 连续收敛（5日趋势递减）
     + 获利比例 < 30%
     + 最近 3 日主力净流入-净额 > 0

含义: 筹码在低位越来越集中 = 有人在悄悄收集
     主力资金净流入证实了"谁"在收集。
     这是经典的「底部吸筹」信号，
     比单纯看资金流可靠得多（筹码提供了价格验证）。
```

#### 信号 ⑤ 获利比例骤升 + 主力不出

```
条件: 获利比例 从 < 30% 快速升至 > 70%（股价拉升中）
     + 主力净流入仍为正

含义: 股价拉升导致大量筹码获利，但主力资金没有外流，
     说明主力还没到出货阶段，上涨行情未结束。
     反过来，如果获利比例 > 70% 且主力开始净流出，
     就是卖出信号了。
```

---

### 三、筹码 + 分时图（intraday 数据）

#### 信号 ⑥ 筹码支撑位 + 分时精确买点

```
条件: 盘中股价回踩「70成本-低」附近（±1%）
     + 分时图出现放量反弹（5分钟成交量 > 前一bar 的 2倍）

含义: 日K级别只能看到收盘价是否触及支撑，
     分时图可以捕捉到盘中精确触及支撑位后的反弹瞬间，
     实现更低成本的买入。
```

#### 信号 ⑦ 平均成本突破 + 尾盘资金脉冲

```
条件: 当日股价从下方上穿「平均成本」
     + 尾盘 30 分钟分时图持续放量上攻

含义: 突破平均成本是日内的关键事件（从套牢变为获利），
     尾盘资金脉冲说明有资金在收盘前抢筹，
     次日高开概率大。
```

---

### 四、筹码 + 龙虎榜（lhb_detail 数据）

#### 信号 ⑧ 筹码集中 + 机构/游资进场

```
条件: 70集中度 < 5%（低位单峰密集）
     + 龙虎榜出现机构专用席位买入
     或知名游资席位买入

含义: 筹码已经在底部充分换手集中，
     龙虎榜大资金进场 = 正式发动的信号。
     没有筹码集中的前提，龙虎榜买入可能只是短线博弈。
```

---

### 五、筹码 + 北向资金（hsgt_hold 数据）

#### 信号 ⑨ 筹码底部 + 北向持续加仓

```
条件: 获利比例 < 20%（低位）
     + 北向持股数量连续 5 日增加

含义: 北向资金通常是中长线逻辑，
     在大部分筹码套牢的低位区间持续加仓，
     说明外资认为当前价格有足够的安全边际。
     筹码低位 + 聪明钱进场 = 高置信度的中线买点。
```

---

### 六、筹码动态变化（纯时序分析）

这类信号不依赖其他数据源，纯粹挖掘筹码自身的时序特征。

#### 信号 ⑩ 筹码搬家：成本重心上移 + 股价滞涨

```
条件: 「平均成本」连续 10 日上移 > 5%
     + 股价涨幅 < 2%

含义: 成本在涨但股价不涨 = 低位筹码被高位资金接走，
     筹码从"老庄"搬到"新庄"手里。
     一旦换手完成（集中度重新收敛），往往有新一轮行情。
```

#### 信号 ⑪ 双峰归一

```
条件: 前期 90成本 区间很宽（90集中度 > 12%）→ 存在套牢峰+获利峰
     + 近期 90集中度 持续缩小至 < 7%

含义: 原来的双峰（上方套牢盘 + 下方获利盘）合并成单峰，
     说明上方套牢盘割肉出局，或下方获利盘加仓拉高成本，
     市场分歧消除，一致性增强，方向选择即将到来。
```

#### 信号 ⑫ 获利比例「黄金交叉」

```
条件: 获利比例 从 < 10% 连续 3 日上升，且斜率加速
     （比如 5% → 12% → 25%）

含义: 类似均线金叉的概念，获利比例加速上穿说明股价
     正在快速脱离底部区域，多头力量在加速释放。
     配合量能放大，可信度更高。
```

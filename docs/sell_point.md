# 持仓卖出决策分析

## 目标

已知个股持仓（代码、名称、买入价、买入日期、持仓数量），结合**市场情绪温度、行业板块热度、龙头股状态**三大宏观/中观分析结果，以及**个股自身的 K 线、成交量、分时图、资金流、实时价格**等微观数据，给出每只持仓股的**量化卖出建议**。

核心理念：**卖出不是孤立的个股决策，而是市场→行业→个股三层联动的综合判断。**

---

## 与现有模块的关系

```
market_temperature（全局情绪 0-100）
    └── industry_temperature（板块热度 + 阶段）
            └── leader_stock（龙头识别 + 评级）
                    └── sell_point（持仓卖出决策）← 本模块
```

本模块**消费**前三个模块的输出，不重复计算。同时叠加个股微观数据做最终决策。

```yaml
    buy_date: "2025-02-01"

```

| 数据 | 来源 | 用途 |

## 分析框架：三层七维度

卖出决策分为**宏观层、中观层、微观层**，共七个分析维度，每个维度输出一个信号和评分。

```
┌────────────────────────────────────────────────────────────────────┐
│                        宏观层（市场环境）                            │
│  维度 1：市场情绪退潮信号                                            │
│  维度 2：系统性风险预警                                              │
├────────────────────────────────────────────────────────────────────┤
│                        中观层（行业/题材）                           │
│  维度 3：行业板块退潮信号                                            │
│  维度 4：龙头地位变化信号                                            │
├────────────────────────────────────────────────────────────────────┤
│                        微观层（个股自身）                            │
│  维度 5：技术形态恶化信号（K 线 + 均线 + 支撑位）                      │
│  维度 6：量价异动信号（成交量 + 分时图 + 大单）                        │
│  维度 7：盈亏管理信号（止损 + 止盈 + 回撤管理）                        │
└────────────────────────────────────────────────────────────────────┘
```

## 第一步：运行上游分析

在分析卖出信号之前，先运行三大上游模块获取市场上下文：

```python
# 1. 刷新数据（全市场 + 持仓股）
dm = DataManager(config)
dm.refresh()

# 2. 运行三大报告
mt = MarketTemperature(dm)
temp_result = mt.run()  # 包含 hot_sectors 和 leader_stocks

# 3. 提取上下文
market_score = temp_result.score         # 市场情绪分数 0-100
market_phase = temp_result.phase         # 冰点期/混沌期/主升期/高潮期
hot_sectors = temp_result.hot_sectors    # List[SectorHeat]
leader_stocks = temp_result.leader_stocks # List[LeaderStock]
```

## 第二步：七维度卖出信号分析

### 维度 1：市场情绪退潮信号（权重 15%）

**数据源**：`TemperatureResult`

市场情绪是卖出决策的"大气候"。情绪退潮时，即使个股基本面不变，也可能被系统性杀跌。

| 检测条件 | 信号 | 卖出评分（0-100） | 说明 |
|---------|------|----------------|------|
| 市场情绪从高潮期（≥80）跌至主升期以下（<50） | 强烈卖出 | 90 | 情绪急转直下，大面风险极高 |
| 市场情绪从主升期（≥50）跌至混沌期（<50） | 建议卖出 | 70 | 赚钱效应消退，资金开始撤退 |
| 市场情绪处于冰点期（<20） | 建议卖出 | 75 | 全市场冰点，覆巢之下无完卵 |
| 市场情绪处于混沌期（20-50） | 关注观望 | 40 | 方向不明，减仓为宜 |
| 市场情绪处于主升期（50-80） | 继续持有 | 10 | 赚钱效应好，持股待涨 |
| 市场情绪处于高潮期（≥80） | 关注观望 | 35 | 物极必反，高潮期后往往退潮，准备兑现 |

**关键指标叠加判断**：

- R_yu（昨日涨停溢价率）< 0%：**接力链断裂**，追板的人在亏钱，所有非龙头股风险骤增 → 卖出评分 +20
- D（跌停数）> 15 且 M（大面数）> 10：**恐慌蔓延** → 卖出评分 +15
- H（最高连板数）较昨日断裂（如 7→4）：**空间龙见顶**，往往引发全市场补跌 → 卖出评分 +15

### 维度 2：系统性风险预警（权重 10%）

**数据源**：`TemperatureMetrics`、全市场行情

检测全市场层面的极端风险事件：

| 检测条件 | 信号 | 卖出评分 |
|---------|------|---------|
| 跌停数 D ≥ 30 | 强烈卖出 | 95 | 
| 大面数 M ≥ 15 | 强烈卖出 | 90 |
| 封板率 FR < 40% 且 涨停数 U < 20 | 建议卖出 | 65 |
| 连板晋级率 PR < 5% | 关注观望 | 45 |
| 无异常 | 继续持有 | 5 |

### 维度 3：行业板块退潮信号（权重 20%）

**数据源**：`List[SectorHeat]`

持仓股所在行业板块的热度变化是最直接的卖出催化剂。板块退潮时，板块内个股（尤其是跟风股）几乎无一幸免。

**Step 3a：匹配持仓股所属行业**

从 `zt_pool` 或 `stock_sector_map` 中查找持仓股的 `所属行业`，与 `hot_sectors` 进行匹配：

| 匹配结果 | 处理 |
|---------|------|
| 持仓股所属行业在 hot_sectors 中找到 | 使用对应 `SectorHeat` 的 phase 和 score |
| 持仓股所属行业不在 hot_sectors 中 | 说明该行业不是当前热点，标记为"冷门行业" |

**Step 3b：根据板块阶段评分**

| 板块阶段 | 信号 | 卖出评分 | 说明 |
|---------|------|---------|------|
| **退潮** | 强烈卖出 | 90 | 资金撤退，龙头断板，跟风股崩塌 |
| **高潮** | 关注观望 | 45 | 全面开花但已到顶部，准备兑现 |
| **高潮**且持仓股不是龙头 | 建议卖出 | 70 | 高潮期跟风股最先被砸，龙头最后倒 |
| **发酵** | 继续持有 | 10 | 板块上升期，持股享溢价 |
| **启动** | 继续持有 | 5 | 板块刚起步，仓位可加 |
| **平稳** | 继续持有 | 15 | 无明显方向 |
| 冷门行业（不在热点中） | 关注观望 | 35 | 非热点行业缺乏资金关注 |

**Step 3c：板块热度骤降叠加**

比较持仓股行业在今日 vs 近期的热度变化（若有历史记录）：
- 行业热度得分较昨日下降 ≥ 30 分 → 卖出评分 +20
- 行业涨停数较昨日减少 ≥ 50% → 卖出评分 +15
- 行业龙头断板（今日连板 < 昨日连板） → 卖出评分 +15

### 维度 4：龙头地位变化信号（权重 15%）

**数据源**：`List[LeaderStock]`

如果持仓股曾经是（或自认为是）板块龙头，需要监控其龙头地位是否松动。

**Step 4a：判断持仓股是否在龙头列表中**

从 `leader_stocks` 中按代码匹配持仓股：

| 匹配结果 | 含义 |
|---------|------|
| 找到且级别为 S/A | 持仓股仍是板块龙头 |
| 找到且级别为 B/跟风 | 持仓股龙头身份存疑 |
| 未找到 | 持仓股不在龙头池中（可能已非热点行业、或连板断裂） |

**Step 4b：龙头状态评分**

| 条件 | 信号 | 卖出评分 | 说明 |
|------|------|---------|------|
| 持仓股为 S 级龙头，板块发酵中 | 继续持有 | 0 | 最强位置，持筹不动 |
| 持仓股为 S 级龙头，板块高潮期 | 关注观望 | 35 | 高潮兑现窗口，关注分歧信号 |
| 持仓股为 A 级龙头 | 继续持有 | 15 | 准龙头，紧密跟踪 |
| 持仓股为 B 级/跟风 | 建议卖出 | 65 | 不是龙头就是跟风，跟风必死 |
| 持仓股不在龙头列表，且连板断裂 | 强烈卖出 | 85 | 龙头身份丧失 |
| 持仓股不在龙头列表，非涨停股 | 关注观望 | 40 | 非热点标的 |
| 同板块出现新的更高连板龙头（龙头交接） | 建议卖出 | 70 | 旧龙头让位，资金转向新龙头 |

**Step 4c：龙头交接检测**

```
if 持仓股连板数 < 板块最高连板数:
    且 板块最高连板股 != 持仓股:
        → "龙头交接"信号，旧龙头应择机兑现
```

**Step 4d：风险信号叠加**

从 `LeaderStock.risk_flags` 获取已有风险标记：
- "连板≥7且板块高潮" → 卖出评分 +10
- "一字板风险（连续一字≥3）" → 卖出评分 +10
- "板块退潮中" → 卖出评分 +20

### 维度 5：技术形态恶化信号（权重 15%）

**数据源**：`daily/{code}.parquet`（日 K 线）

对持仓股自身的技术面进行评估。分析买入日到当前的完整 K 线走势。

**5a. 均线系统（0-30 分）**

计算 MA5、MA10、MA20、MA60：

| 条件 | 卖出评分 |
|------|---------|
| 收盘 < MA60 且 MA5 < MA20（趋势彻底破坏） | 90 |
| 收盘 < MA20 且 MA5 < MA10（中期空头） | 70 |
| 收盘 < MA20（跌破中期均线，待确认） | 45 |
| MA5 下穿 MA10（短期死叉） | 30 |
| MA5 > MA10 > MA20（多头排列） | 5 |

**5b. 关键支撑位（0-25 分）**

识别买入以来的关键支撑/阻力位：

```
前期低点 = 买入日以来日K最低价
前期高点 = 买入日以来日K最高价
半分位 = (前期高点 + 前期低点) / 2
```

| 条件 | 卖出评分 |
|------|---------|
| 当前价跌破前期低点（支撑位完全失守） | 85 |
| 当前价跌破半分位且放量 | 60 |
| 当前价接近前期高点（压力位附近） | 15 |

**5c. K 线形态（0-20 分）**

识别近 3-5 日的典型见顶 K 线组合：

| K 线形态 | 卖出评分 | 识别方法 |
|---------|---------|---------|
| 长上影线（上影 ≥ 实体 2 倍） | 50 | `(最高 - max(开盘,收盘)) ≥ 2 × abs(收盘-开盘)` |
| 十字星（实体极小 + 上下影线长） | 35 | `abs(收盘-开盘)/开盘 < 0.5%` 且上下影均 > 1% |
| 吞没阴线（大阴线吃掉前一日阳线） | 65 | 今日跌幅 > 3% 且 今日收盘 < 昨日开盘 |
| 连续阴线 ≥ 3 天 | 55 | 近 3 日收盘均 < 开盘 |
| 跳空低开（缺口未回补） | 60 | 今日开盘 < 昨日最低 |
| 无明显见顶形态 | 5 | — |

**5d. 持仓天数与波动率**

```
持仓天数 = 买入日至今的交易日天数
期间波动率 = 日收益率标准差 × √252（年化）
```

| 条件 | 卖出评分调整 |
|------|------------|
| 持仓 > 20 天且横盘（涨跌幅 < 5%） | +10（长期横盘消耗时间成本） |
| 持仓 > 30 天且亏损 | +15（深套久盘必跌） |
| 波动率 > 80%（年化）且亏损 | +10（高波动 + 亏损 = 高风险） |

### 维度 6：量价异动信号（权重 15%）

**数据源**：日 K 线 + 分时图（`intraday`）+ 资金流（`fund_flow`）+ 大单追踪（`big_deal`）

**6a. 成交量分析（0-30 分）**

```
量比 = 今日成交量 / 近10日平均成交量
换手率 = 今日成交量 / 流通股本
```

| 条件 | 卖出评分 |
|------|---------|
| 放量大跌（量比 > 2 且跌幅 > 3%）：恐慌出逃 | 80 |
| 量价顶背离（价格创新高但量能逐日递减 ≥ 3 天） | 70 |
| 放量滞涨（量比 > 1.5 且近 3 日涨跌 < 2%）：主力出货 | 65 |
| 天量换手（换手率 > 15%）：资金分歧极大 | 55 |
| 极度缩量上涨（量比 < 0.5 且上涨）：动能衰减 | 30 |
| 缩量整理（量比 0.5-1.0，小幅波动） | 10 |
| 量价配合良好 | 5 |

**6b. 分时图分析（0-25 分）**

从 `intraday/{code}.parquet`（5 分钟 K 线）提取当日分时特征：

```
日内振幅 = (当日最高 - 当日最低) / 昨收
尾盘走势 = 14:00-15:00 期间的涨跌幅
早盘走势 = 09:30-10:30 期间的涨跌幅
```

| 条件 | 卖出评分 | 说明 |
|------|---------|------|
| 高开低走（早盘涨 > 2%，尾盘收跌） | 60 | 主力拉高出货 |
| 尾盘急跌（14:30后跌幅 > 2%） | 55 | 资金尾盘砸盘 |
| 脉冲式下跌（分时出现连续放量大阴柱） | 50 | 大单集中出逃 |
| 日内振幅 > 10% 且收阴 | 45 | 极端分歧后空头胜出 |
| 全天低位横盘（分时低于均价线运行） | 35 | 空头主导 |
| 分时正常波动 | 5 | — |

**6c. 资金流向分析（0-25 分）**

从 `fund_flow/{code}` 获取主力/大单净流入数据：

| 条件 | 卖出评分 | 说明 |
|------|---------|------|
| 连续 3 日主力净流出，累计流出 > 5000 万 | 75 | 主力系统性撤退 |
| 连续 3 日超大单净流出 | 60 | 顶级资金在撤退 |
| 今日主力净流出 > 1 亿 | 55 | 单日巨额流出 |
| 近 5 日主力累计净流出 > 3000 万 | 45 | 资金持续流出 |
| 近 5 日主力累计净流入 > 5000 万 | 5 | 资金持续流入，安全 |
| 资金流中性 | 15 | — |

**6d. 大单追踪（0-20 分）**

从 `big_deal` 中筛选持仓股的大单成交记录：

| 条件 | 卖出评分 | 说明 |
|------|---------|------|
| 当日出现多笔大额卖单（单笔 > 500 万），卖出金额 > 买入金额 2 倍 | 65 | 大资金集中出逃 |
| 大额买单集中在早盘、大额卖单集中在尾盘 | 55 | 典型拉高出货形态 |
| 大单买卖平衡 | 10 | 正常换手 |
| 无大单数据 | 0 | 数据缺失，不计分 |

### 维度 7：盈亏管理信号（权重 10%）

**数据源**：买入价、当前价、日 K 线

纯粹基于盈亏比例和风险管理纪律的硬性规则。

**7a. 硬止损（最高优先级）**

| 条件 | 信号 | 卖出评分 |
|------|------|---------|
| 浮亏 ≥ 8% | 强烈卖出 | 95 |
| 浮亏 5%-8% | 建议卖出 | 70 |
| 浮亏 2%-5% | 关注观望 | 35 |

**7b. 移动止盈（回撤管理）**

```
持仓期最高价 = daily[买入日:今日]['最高'].max()
回撤幅度 = (持仓期最高价 - 当前价) / 持仓期最高价 × 100%
```

回撤止盈的阈值根据已盈利幅度动态调整——**赚得越多，止盈越宽松**：

| 当前盈利幅度 | 回撤止盈触发阈值 | 说明 |
|------------|----------------|------|
| 盈利 > 50% | 回撤 25% 触发 | 大赚时给足空间 |
| 盈利 20%-50% | 回撤 15% 触发 | 中等盈利，适度保护 |
| 盈利 10%-20% | 回撤 10% 触发 | 小赚要守住 |
| 盈利 < 10% | 回撤 8% 触发 | 刚有浮盈，严格保护 |

| 条件 | 信号 | 卖出评分 |
|------|------|---------|
| 触发回撤止盈 | 建议卖出 | 75 |
| 接近回撤止盈（差 2% 以内） | 关注观望 | 45 |

**7c. 盈利兑现参考**

| 条件 | 卖出评分 |
|------|---------|
| 盈利 ≥ 50% 且持仓 > 10 天 | 40（关注兑现时机） |
| 盈利 ≥ 30% | 25（关注止盈） |
| 盈利 10%-30% | 10（持股待涨） |
| 微利或亏损 | 0 |

## 第三步：综合评分与决策

### 3.1 加权合成

$$Score_{卖出} = \sum_{i=1}^{7} w_i \times S_i$$

各维度权重：

| 维度 | 权重 |
|------|------|
| 维度 1：市场情绪退潮 | 15% |
| 维度 2：系统性风险 | 10% |
| 维度 3：行业板块退潮 | 20% |
| 维度 4：龙头地位变化 | 15% |
| 维度 5：技术形态恶化 | 15% |
| 维度 6：量价异动 | 15% |
| 维度 7：盈亏管理 | 10% |

> **权重设计理念**：行业板块退潮（20%）权重最高，因为短线股最受板块共振影响；市场情绪（15%）+ 龙头地位（15%）决定大方向；个股技术（15%）+ 量价（15%）提供精确时点；盈亏管理（10%）是纪律底线。

### 3.2 卖出信号分级

| 综合评分 | 信号 | 建议操作 |
|---------|------|---------|
| ≥ 80 | 🔴 **强烈卖出** | 立即清仓，不犹豫 |
| 60-80 | 🟠 **建议卖出** | 择机卖出，至少减半仓 |
| 40-60 | 🟡 **关注观望** | 设好止损，密切关注，准备随时撤退 |
| 20-40 | 🟢 **继续持有** | 安全持股，无需操作 |
| < 20 | 🔵 **可加仓** | 多头信号强，可考虑加仓 |

### 3.3 硬性规则（一票否决）

以下条件触发时，无论综合评分多低，直接升级为对应信号：

| 条件 | 直接信号 | 理由 |
|------|---------|------|
| 浮亏 ≥ 8% | 🔴 强烈卖出 | 硬止损铁律，不可违反 |
| 板块阶段 = "退潮" 且持仓股不是 S 级龙头 | 🟠 建议卖出 | 退潮板块跟风股必死 |
| 市场情绪 < 15 且 R_yu < -1% | 🟠 建议卖出 | 市场冰点 + 接力断裂 |
| 跌停 | 🔴 强烈卖出（若能挂单） | 止损最优先 |

### 3.4 加分规则（利好叠加）

以下条件触发时，降低卖出评分（鼓励持有）：

| 条件 | 卖出评分调整 |
|------|------------|
| 持仓股为 S 级龙头，板块发酵/主升期 | -20 |
| 市场情绪 > 60 且 R_yu > 2% | -10 |
| 今日涨停封板（封板资金排名前 10） | -15 |
| 北向资金今日大幅净买入（> 50 亿） | -5 |
| 板块出现新催化（概念板块涨幅 > 5%） | -10 |

## 第四步：关联股票分析

对每只持仓股，额外分析同板块其他关键股票的表现，作为辅助判断：

### 4.1 板块龙头状态

```
if 持仓股 != 板块龙头:
    龙头今日表现 → 龙头涨停 = 安全信号，龙头炸板/下跌 = 危险信号
    龙头连板是否继续 → 断板 = 板块可能退潮
```

### 4.2 板块跟风股表现

```
同板块涨停数变化：
  今日涨停数 < 昨日涨停数 × 0.5 → 板块正在退潮
  今日涨停数 > 昨日涨停数 × 1.5 → 板块正在发酵
```

### 4.3 相关概念股

从 `concept_board` 中找到持仓股可能关联的概念板块：
- 如果关联概念板块今日涨幅靠前 → 利好叠加
- 如果关联概念板块今日领跌 → 利空叠加

### 4.4 竞品/替代标的

在同行业中，如果出现新的更强标的（首板/2板，封单更大，市值更合适），资金可能从老龙头切换到新龙头：
- 新龙头涨停数 > 旧龙头 → "资金切换"警告

## 注意事项

1. **数据刷新策略**：`cmd_sell` 需要 `dm.refresh()`（全量刷新）而非 `dm.refresh_for_stocks()`，因为三大上游报告需要全市场数据（涨停池、行业资金流等）。持仓股的个股数据（日K、分时、资金流）在全量刷新中已包含。

2. **交易时段差异**：
   - **盘中运行**（9:30-15:00）：实时价格可用，分时图持续更新，信号更精准
   - **收盘后运行**（15:00 后）：使用收盘数据，输出的是"次日操作建议"
   - **非交易时段**：回退到最近交易日数据

3. **持仓股不在涨停池**：大部分持仓股可能不在 `zt_pool` 中（不是涨停股），此时维度 4（龙头地位）评分偏中性。只有当持仓股曾经是龙头但掉出龙头池时，才产生卖出信号。

4. **行业匹配精度**：`stock_sector_map` 和 `zt_pool.所属行业` 的行业名可能不完全覆盖所有持仓股。对于无法匹配行业的持仓股，维度 3 和维度 4 给予中性评分（不加分不扣分）。

5. **分时数据精度**：分时数据为 5 分钟级别，对于日内快速波动的判断有延迟。尾盘急跌等判断基于最后 6 个 5 分钟 K 线。

6. **资金流数据覆盖**：`fund_flow` 数据可能不覆盖所有持仓股（特别是冷门小盘股）。缺失时该维度默认中性评分。

7. **硬止损不可商量**：浮亏 ≥ 8% 的硬止损规则是"铁律"——即使市场情绪高涨、板块发酵中，也要执行止损。短线交易的第一原则是**保住本金**。亏 8% 后需要涨 8.7% 才能回本；亏 20% 则需要涨 25%。亏损越深，翻身越难。

8. **高潮期的特殊处理**：市场情绪 ≥ 80（高潮期）不是"安全"信号，反而是危险信号。高潮之后往往是快速退潮，需要提前准备兑现计划。报告中应明确提示"高潮期持筹不追高，准备冲高兑现"。

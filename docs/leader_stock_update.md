# 龙头股买入增强 — 历史K线 + 筹码辅助判断

## 一、当前 leader_stock 的盲区

现有龙头打分的五个维度（连板、先手、封板、辨识、助攻）全部是**"当下快照"**——只看涨停池数据和当天的市场行为。它回答的是"这只股票现在看起来像龙头"，但不回答以下关键问题：

| 问题 | 当前能回答吗 | 历史K线能回答 |
|------|-------------|-------------|
| 起涨位置在哪里？是底部启动还是高位加速？ | ❌ | ✅ |
| 涨停前涨了多少？已经翻倍了还是刚起步？ | ❌ | ✅ |
| 中期趋势是上升还是下降中的反弹？ | ❌ | ✅ |
| 涨停前有没有经历充分的横盘整理？ | ❌ | ✅ |
| 上方有没有密集的套牢区压力？ | ❌ | ✅ |
| 这只股票历史上当过龙头吗？结局如何？ | ❌ | ✅ |

一个极端例子：A 股和 B 股今天都是 3 连板、板块第一，龙头评分完全相同。但 A 股是从底部横盘 3 个月后首次启动，B 股是从高位已经翻了一倍后继续加速。两者的买入价值天差地别——历史 K 线是唯一能区分它们的工具。

## 二、历史K线辅助维度

### 维度 1：起涨位置（安全边际）

```
看什么: 第一个涨停板相对于近 60 日最低价的位置
指标:   涨停起点涨幅 = (首板价 - 60日最低价) / 60日最低价

低位启动 (< 15%):  安全边际高，即使龙头失败回落空间有限
中位启动 (15%~40%): 正常，需配合其他确认
高位启动 (> 40%):  追高风险大，一旦断板可能暴跌
已翻倍启动 (> 100%): 极高风险，除非是超级龙头（年度级别题材）
```

这是决定"能不能买"的第一道门槛。

### 维度 2：启动前形态（蓄力程度）

涨停前的 K 线形态决定了行情的"能量储备"：

| 形态 | K线特征 | 对龙头的意义 |
|------|---------|-------------|
| **横盘整理后突破** | 20~60 日窄幅震荡，振幅 < 15%，然后涨停 | ★★★★★ 最佳。充分换手+蓄力，突破后空间大 |
| **底部圆弧底** | 缓慢下跌→企稳→缓慢抬升→涨停 | ★★★★ 趋势反转确认，中线行情 |
| **急跌后V型反弹** | 连续暴跌后直接涨停 | ★★★ 超跌反弹，空间看跌幅深度，持续性存疑 |
| **上升途中加速** | 已沿 MA20 上行一段时间，突然加速涨停 | ★★★ 趋势继续，但要看前期涨幅 |
| **高位横盘后再突破** | 已大涨后高位盘整再涨停 | ★★ 可能是最后一冲，警惕出货 |
| **无明显形态** | 随机波动中突然涨停 | ★★ 可能是消息驱动，持续性差 |

### 维度 3：均线系统（趋势上下文）

```
MA5 > MA10 > MA20 > MA60（多头排列）:
  → 中期趋势向上，龙头涨停是顺势加速，可信度+30%

MA20 走平，MA60 仍向下:
  → 中期趋势未确认反转，龙头可能只是反弹中的脉冲，持续性打折

MA20 < MA60 且均下行（空头排列）:
  → 逆势涨停，风险极高。除非是超强题材的"逆势龙"，否则不追
```

### 维度 4：前期涨幅与空间（天花板判断）

```
看什么: 从近 120 日最低点到当前价的累计涨幅
       + 当前价距近 120 日最高点的距离

累计涨幅 < 30% 且距前高 > 20%:
  → 上方空间大，套牢压力小，买入安全边际充足

累计涨幅 50%~100% 且接近前高:
  → 前高是关键压力位，能否突破决定后续空间
  → 需要结合筹码判断前高附近套牢盘密度

累计涨幅 > 100%:
  → 空间可能有限，追高风险大，仓位必须控制
```

### 维度 5：量价历史（资金行为轨迹）

```
启动前缩量 → 涨停放量:
  → 经典的"缩量蓄力+放量突破"，健康

启动前就持续放量 → 涨停继续放量:
  → 可能是资金提前介入（利好预期），关注量能能否持续

涨停时量能是近 20 日均量的 3 倍以上:
  → 巨量涨停，可能封不住或次日分歧大

涨停时量能仅为均量的 1~2 倍:
  → 轻松涨停，说明筹码锁定好，后续连板概率高
```

### 维度 6：历史龙头记忆（市场辨识度加成）

```
看什么: 过去 1 年内该股是否出现过连板（>=3板）

有过成功龙头经历:
  → 市场对它有"龙头记忆"，游资愿意给更高的辨识溢价
  → 典型案例: 反复成为妖股的标的，每次板块起来都被第一时间联想

从未连板过:
  → 市场对它没有预期，需要更强的板块催化才能被认可
  → 首次成为龙头时，前 1~2 板市场将信将疑，资金合力弱
```

## 三、筹码确认维度

结合 leader_stock 的 `lifecycle` 字段，筹码在不同生命周期下含义不同：

| 龙头生命周期 | 筹码状态 | 组合含义 | 操作价值 |
|-------------|---------|---------|---------|
| **启动期** + 单峰密集突破 | 70集中度 < 4%，获利比例从极低开始拉升 | **最佳买点**。主力吸筹完毕、筹码锁定、板块刚启动、市场开始辨识。这是龙头行情的"第一买点" | ★★★★★ |
| **加速期** + 单峰仍在 | 70集中度仍 < 5%，获利比例 50%~80% | **加仓窗口**。虽然已涨了一段，但筹码没散说明主力没出，加速阶段利润最丰厚 | ★★★★ |
| **高潮期** + 筹码开始发散 | 70集中度 > 8%，90集中度 > 12% | **警告信号**。龙头仍有人气但筹码已经从单峰变双峰，高位换手剧烈 = 主力在分批出货。此时"放量突破"大概率是**诱多** | ⚠️ 不买 |
| **退潮期** + 单峰密集 | 集中度重新收敛，但在低位 | **需要等待**。龙头退潮后筹码在低位重新集中 = 新一轮吸筹可能开始，但需要等板块重新启动确认 | ★★（观察） |

筹码还能帮助区分真伪龙头：

| 龙头类型 | 筹码特征 | 判断 |
|----------|---------|------|
| 真龙头 | 启动前筹码就已集中（70集中度 < 5%），突破后集中度不变或更低 | 有主力长期运作，可信 |
| 伪龙头（一日游） | 筹码分散（70集中度 > 10%），靠单日游资拉板 | 没有筹码基础，次日大概率分歧大开，回落 |
| 接力龙头 | 前龙头退潮时该股筹码已在低位集中 | 有可能成为新龙头，值得关注 |

## 四、综合判断矩阵

把历史K线的六个维度浓缩为一个"位置质量"评分，与现有龙头评级交叉：

| | 龙头评级 S/A | 龙头评级 B | 龙头评级 C/D |
|---|---|---|---|
| **K线位置优（低位+横盘突破+多头排列）** | 最强买入信号，可重仓 | 强买入，正常仓位 | 观察，等升级 |
| **K线位置中（中位+趋势向上）** | 买入，正常仓位 | 买入，轻仓 | 不操作 |
| **K线位置差（高位+已翻倍+空头排列）** | 谨慎，极轻仓或不追 | 不操作 | 不操作 |

核心结论：**龙头评级决定"要不要关注"，历史K线决定"能不能买、买多少"**。两者缺一不可。

# 日K形态检测 — 连续评分函数设计

## 设计原则

不使用硬阈值（如"振幅 < 15%"、"天数 >= 20"），而是将每个因子映射到 `[0, 1]` 的连续分数，再加权组合。

**原因**：
1. **避免悬崖效应**：振幅 14.9% 得满分，15.1% 得零分，差 0.2% 却结果迥异
2. **适应个股差异**：银行股日常振幅 5%，科技股日常振幅 20%，固定阈值无法兼顾
3. **区分质量**：连续评分能区分"还行"和"极佳"的形态，而非一刀切的"有/没有"

---

## 形态 ①：横盘整理后突破

### 因子拆解

横盘突破的本质是四件事同时发生：**波动收敛** + **持续够久** + **价格突破** + **量能确认**。

### 因子 1：波动收敛度（权重 0.35）

用该股自身的历史波动率做基准，而非绝对振幅阈值。

```
计算:
  σ_recent  = std(近 N 日收益率)        # 近期波动率（N = 横盘窗口）
  σ_history = std(前 60~120 日收益率)   # 该股自身基准波动率

  VR = σ_recent / σ_history             # 波动率比值

  VR < 0.3 → 极度收敛（对这只股票来说非常安静）
  VR ≈ 0.5 → 明显收敛
  VR ≈ 1.0 → 正常波动，不算横盘
  VR > 1.0 → 波动放大，完全不是横盘

映射函数（平滑幂衰减）:
  f_vol = max(0, 1 - VR) ^ 0.6

  VR=0.2 → f_vol=0.88
  VR=0.4 → f_vol=0.69
  VR=0.6 → f_vol=0.47
  VR=0.8 → f_vol=0.22
  VR=1.0 → f_vol=0
```

**为什么用相对波动率**：银行股 σ_history 可能只有 1.2%，近期 σ_recent=0.6% 则 VR=0.5，说明波动收敛了一半，是有意义的横盘。科技股 σ_history 可能有 3.5%，同样 σ_recent=0.6% 则 VR=0.17，说明极度收敛。两者在各自尺度下都得到合理评价。

### 因子 2：横盘持续时间（权重 0.25）

不设硬性最低天数，时间越长分越高，但有饱和上限。

```
映射函数（对数增长，自然饱和）:
  f_dur = min(1, ln(days / 10) / ln(6))

  10日 → f_dur=0.00（刚满最低观察期，不加分）
  15日 → f_dur=0.23
  20日 → f_dur=0.39
  30日 → f_dur=0.61
  45日 → f_dur=0.84
  60日 → f_dur=1.00（饱和，更长也不额外加分）
```

**为什么用对数函数**：横盘从 10 天到 30 天的信息增量很大（确认了主力在吸筹），但从 30 天到 60 天的增量递减（已经充分确认了）。对数函数天然具备这种"前快后慢"的增长特性。

### 因子 3：突破强度（权重 0.25）

不仅看"是否突破窗口最高价"，还看突破了多少。

```
计算:
  breakout = (涨停当日收盘 - 窗口最高价) / 窗口最高价

映射函数（线性截断）:
  f_brk = clip(breakout / 0.08, 0, 1)

  breakout=0%  → f_brk=0.00（刚好触及，未突破）
  breakout=2%  → f_brk=0.25
  breakout=4%  → f_brk=0.50
  breakout=8%+ → f_brk=1.00（涨停突破，满分）
```

**为什么 8% 封顶**：主板涨停幅度为 10%，突破窗口高点 8% 以上基本意味着涨停突破，再高也没有额外信息量。

### 因子 4：量能确认（权重 0.15）

突破当日的成交量相对于横盘期间均量的放大倍数。

```
计算:
  vol_ratio = 涨停当日成交量 / MA(成交量, 窗口N日)

映射函数（线性截断）:
  f_vol_confirm = clip((vol_ratio - 1.0) / 2.0, 0, 1)

  vol_ratio=1.0 → f_vol_confirm=0.00（不放量）
  vol_ratio=1.5 → f_vol_confirm=0.25
  vol_ratio=2.0 → f_vol_confirm=0.50
  vol_ratio=3.0 → f_vol_confirm=1.00（3倍量，满分）
```

**权重最低的原因**：有些缩量突破也有效（主力高度控盘时不需要放量），量能是辅助确认而非必要条件。

### 组合公式

```
横盘突破分 = 20 × (0.35·f_vol + 0.25·f_dur + 0.25·f_brk + 0.15·f_vol_confirm)

得分范围: 0 ~ 20（连续值）
```

### 窗口自适应

不固定窗口长度，而是自动搜索最优窗口：

```
for N in range(15, 61):
    计算以涨停前 N 日为窗口的 f_vol
    if f_vol > 0.3:  # 波动率有收敛
        计算该窗口下的完整得分
记录所有窗口中得分最高的作为最终结果
```

### 效果对比

| 场景 | 硬阈值方案 | 连续评分方案 |
|------|-----------|-------------|
| 横盘 25 日，振幅 14%，放量突破 | +20（全部命中） | 约 +16（各因子都不错但未极致） |
| 横盘 25 日，振幅 16%，放量突破 | **0**（振幅超阈值，全否） | 约 +12（振幅略大但其他因子好） |
| 横盘 18 日，振幅 10%，放量突破 | **0**（天数不够，全否） | 约 +11（天数稍短但收敛极强） |
| 横盘 50 日，振幅 8%，3 倍量突破 | +20（和 25 日一样） | 约 +18（更久更收敛，得分更高） |
| 横盘 10 日，振幅 20%，放量 | 0 | 约 +2（几乎不算横盘） |

## 形态 ②：底部圆弧底

### 因子拆解

圆弧底的本质是：**前期下跌** + **中段筑底** + **后期回升** + **弧形平滑**。

### 因子 1：弧形程度（权重 0.35）

衡量价格曲线的"U 型"程度。

```
计算:
  取涨停前 40~90 日的日K，等分为三段
  avg_1 = 前段收盘价均值
  avg_2 = 中段收盘价均值
  avg_3 = 后段收盘价均值

  dip_left  = (avg_1 - avg_2) / avg_1    # 左侧下降深度
  rise_right = (avg_3 - avg_2) / avg_2    # 右侧回升幅度

  arc_score = min(dip_left, rise_right) / max(dip_left, rise_right)
  # 左右越对称，arc_score 越接近 1

映射函数:
  f_arc = sqrt(min(dip_left, 0.10) / 0.10) × sqrt(min(rise_right, 0.10) / 0.10) × arc_score

  完美圆弧（左右各跌/涨 10%，对称）→ f_arc ≈ 1.0
  只跌不升（V 型左半边）→ rise_right ≈ 0 → f_arc ≈ 0
  左右严重不对称 → arc_score 低 → f_arc 打折
```

### 因子 2：底部持续时间（权重 0.25）

```
计算:
  bottom_days = 中段天数（价格在 avg_2 ± 3% 范围内的连续交易日）

映射函数:
  f_bottom_dur = min(1, ln(bottom_days / 8) / ln(5))

  8日  → 0.00
  15日 → 0.40
  25日 → 0.71
  40日 → 1.00
```

### 因子 3：趋势反转确认（权重 0.25）

```
计算:
  后段的 MA5 与 MA20 的关系
  slope_ma20 = (后段末尾MA20 - 后段起始MA20) / 后段天数

映射函数:
  if MA5 > MA20 且 slope_ma20 > 0:
    f_trend = clip(slope_ma20 / 0.005, 0, 1)
  else:
    f_trend = 0
```

### 因子 4：涨停位置（权重 0.15）

涨停发生在圆弧底的右侧上升段末端时最有意义。

```
计算:
  pos = 涨停当日在整个窗口中的位置百分比（0=最左，1=最右）

映射函数:
  f_pos = clip((pos - 0.6) / 0.3, 0, 1)

  涨停在窗口 60% 位置 → 0.00（太早，还在底部）
  涨停在窗口 75% 位置 → 0.50
  涨停在窗口 90% 位置 → 1.00（右侧上升末端，最佳）
```

### 组合公式

```
圆弧底分 = 15 × (0.35·f_arc + 0.25·f_bottom_dur + 0.25·f_trend + 0.15·f_pos)

得分范围: 0 ~ 15（连续值）
```

### 窗口自适应

```
for total_days in range(40, 91, 5):
    将窗口等分为三段，计算各因子
    记录得分最高的窗口长度作为最终结果
```

## 通用环境因子

以下三个因子独立于具体形态，适用于所有候选股。它们作为环境修正叠加在形态分之上。

### 环境因子 H1：均线系统评分（-5 ~ +5）

均线排列是趋势的"大背景"。同一个横盘突破，发生在多头排列和空头排列中含义完全不同。

```
计算:
  ma5  = MA(收盘, 5)
  ma10 = MA(收盘, 10)
  ma20 = MA(收盘, 20)
  ma60 = MA(收盘, 60)

  slope_ma20 = (ma20_today - ma20_5days_ago) / ma20_5days_ago  # MA20 近5日斜率
  slope_ma60 = (ma60_today - ma60_5days_ago) / ma60_5days_ago  # MA60 近5日斜率
```

**四档评分**：

```
完全多头排列 (ma5 > ma10 > ma20 > ma60 且 slope_ma20 > 0):
  f_ma = +5
  理由: `均线完全多头排列，中期趋势向上`

部分多头 (ma5 > ma20 且 slope_ma20 > 0，但 ma20 < ma60):
  f_ma = +2
  理由: `短期均线多头，中期趋势待确认`

均线粘合 (ma5, ma10, ma20 之间偏离 < 2%):
  f_ma = 0
  理由: `均线粘合，方向未明`

空头排列 (ma5 < ma10 < ma20 且 slope_ma20 < 0):
  f_ma = -5
  理由: `均线空头排列，逆势涨停风险高`
```

**为什么独立评分**：均线是所有技术分析的基础语境。横盘突破 +18 分 + 空头排列 -5 分 = +13 分，自然降权了逆势突破；而横盘突破 +18 分 + 多头排列 +5 分 = +23 分，顺势突破得到合理加强。不需要在每个形态内部重复处理均线逻辑。

---

### 环境因子 H2：量价趋势配合度（-5 ~ +5）

不看单日量比，而是看**过去一段时间**量和价的配合趋势。核心判断：涨的时候是否放量、跌的时候是否缩量。

```
计算（取近 20 个交易日）:
  将每日标记为"上涨日"（收盘 > 开盘）或"下跌日"（收盘 <= 开盘）

  avg_vol_up   = 上涨日的平均成交量
  avg_vol_down = 下跌日的平均成交量

  vp_ratio = avg_vol_up / avg_vol_down

含义:
  vp_ratio > 1 → 涨时放量跌时缩量，资金在主动买入（健康）
  vp_ratio < 1 → 涨时缩量跌时放量，资金在主动卖出（危险）

映射函数:
  f_vp = clip((vp_ratio - 1.0) / 0.8, -1, 1) × 5

  vp_ratio=1.8+ → f_vp=+5（涨时量远大于跌时量，极度健康）
  vp_ratio=1.4  → f_vp=+2.5
  vp_ratio=1.0  → f_vp=0（涨跌量持平，中性）
  vp_ratio=0.6  → f_vp=-2.5
  vp_ratio=0.2- → f_vp=-5（涨时几乎无量、跌时巨量，出货特征）
```

**为什么重要**：横盘期间如果量价配合度高（vp_ratio > 1.3），说明横盘是"主力吸筹"——每次小涨都有资金进场，每次小跌都缩量惜售。反之如果 vp_ratio < 0.8，同样的横盘更可能是"主力出货"——每次反弹缩量无人跟，每次下跌放量有人卖。**同样的形态，量价配合度不同，含义截然相反。**

---

### 环境因子 H3：关键价位突破加分（0 ~ +5）

突破的不仅是横盘窗口最高价，如果同时穿越了市场公认的关键价位，信号强度倍增。

```
检测的关键价位（按重要性排序）:
  1. 前期高点: 近 120 日最高收盘价
  2. 长期均线: MA120 或 MA250
  3. 中期均线: MA60

逐一检查涨停当日是否同时突破:
  score = 0
  if 昨收 < 前期高点 且 今收 > 前期高点:     score += 3
  if 昨收 < MA120   且 今收 > MA120:         score += 2
  elif 昨收 < MA60  且 今收 > MA60:          score += 1

  f_key = min(score, 5)
```

| 突破情况 | 分数 | 理由模板 |
|----------|------|---------|
| 同时突破前高 + MA120 | +5 | `涨停同时突破120日前高({price})和MA120，多重压力位一举攻克` |
| 突破前期高点 | +3 | `涨停突破120日前高({price})，新高确认` |
| 突破 MA120 | +2 | `涨停站上MA120({price})，长期趋势转多` |
| 突破 MA60 | +1 | `涨停站上MA60({price})，中期趋势向好` |
| 未突破任何关键价位 | 0 | — |

**为什么重要**：关键价位是全市场技术派的共识阻力位。当一根涨停K线同时突破前高和长期均线时，会触发大量技术派的跟风买单和空头的止损买入，形成自我强化的上涨动力。这种突破比单纯突破横盘区间要强得多。

## 最终 K 线总分

```
K线总分 = 形态分(F1) + 环境因子(H1 + H2 + H3)

其中:
  F1 = max(横盘突破分, 圆弧底分, 0) + 负面形态扣分    范围: -10 ~ +20
  H1 = 均线系统评分                                   范围: -5 ~ +5
  H2 = 量价趋势配合度                                 范围: -5 ~ +5
  H3 = 关键价位突破加分                               范围:  0 ~ +5

K线总分范围: -20 ~ +35
```

在 `leader_stock_v1.md` 中，K线总分替代原来的"形态加分(F1+F2)"，与龙头身份分、筹码分共同构成最终总分。

## 其他形态的惩罚分

非横盘突破和圆弧底的形态，取两者得分的较大值。如果两者都很低（< 3 分），说明没有可识别的蓄力形态，再检查是否存在负面形态：

| 负面形态 | 检测方法 | 分数 |
|----------|---------|------|
| 高位横盘后再突破 | rise_from_120d_low > 80% 且横盘突破分 > 5 | 横盘突破分 × (-0.5)，即反扣 |
| 连续急跌后涨停 | 前 5 日累计跌幅 > 15% | -5 |

```
最终形态分 F1 = max(横盘突破分, 圆弧底分, 0) + 负面形态扣分
范围: -10 ~ +20
```

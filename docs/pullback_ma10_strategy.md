# 多头排列回踩MA10缩量阴线 — 趋势回踩买入策略

## 一、策略核心理念

### 1.1 交易什么

做「多头趋势确立后的缩量回踩」——在一轮上涨行情中，等价格回落到 MA10 附近、缩量收阴的那一刻介入。

上涨趋势中的价格运动不是直线拉升，而是「上涨 → 回踩 → 再上涨」的阶梯式推进：

```
      ╱‾‾╲                 ← 第二波放量上涨
     ╱    ╲   ╱‾‾‾‾
    ╱      ╲_╱             ← 缩量回踩MA10（本策略买点）
   ╱
  ╱                        ← 第一波放量上涨
─╱
```

大多数人在放量拉升时追涨、在缩量回调时恐慌卖出。本策略反过来：**在别人恐慌的缩量阴线上介入，在回踩支撑位时拿到更低的成本**。

### 1.2 预期差视角

> **市场认为缩量阴线 = 走弱信号，实际上 = 主力洗盘、获利盘清洗完毕。**

当一只正处于多头趋势中的股票收出缩量阴线、价格回落到 MA10 附近时：
- **缩量**说明卖出意愿不强，主力并没有离场
- **阴线**让短线客和获利盘恐慌出局，正好完成洗盘
- **MA10 支撑**说明中短期成本线仍然有效
- **过去的放量上涨**证明这只股票有资金关注、有上攻动能

四个要素叠加在一起，本质是：**趋势没变、筹码洗干净了、价格到了一个舒适的上车位**。


## 二、选股条件

### 2.1 四大核心条件

同时满足以下四个条件，才视为有效的「回踩买入信号」：

#### 条件 1：均线多头排列 + 斜率向上且温和

四条均线严格从上到下排列，方向一致向上，说明短中期资金持续做多。但斜率不能过陡——斜率太大意味着短线涨幅过急，回踩可能较深。MA5 和 MA10 都需要控制斜率上限。

```
排列要求:
  MA5_T > MA10_T > MA20_T > MA30_T

斜率要求 1 — MA10（中短期趋势锚点）:
  slope_ma10 = (MA10_T - MA10_(T-5)) / MA10_(T-5)

  0.5% <= slope_ma10 <= 4.0%

  含义:
    < 0.5%  — 均线虽然排列，但几乎走平，趋势动能不足
    0.5%~2.0% — 温和上行，最理想的状态（稳健牛）
    2.0%~4.0% — 中等上行，可参与但需注意回踩幅度
    > 4.0%  — 急涨陡峭，回踩风险大，不入选

斜率要求 2 — MA5（短线趋势锚点）:
  slope_ma5 = (MA5_T - MA5_(T-3)) / MA5_(T-3)

  slope_ma5 <= 6.0%

  含义:
    < 0%    — MA5 已拐头向下（回踩阴线当日可能出现，
              只要 MA10 仍向上且 MA5 > MA10 则可容忍，见评分细则）
    0%~2.0% — 温和上行（最理想，说明上涨节奏平稳）
    2.0%~4.0% — 中等上行（正常）
    4.0%~6.0% — 偏快（可接受但扣分，近期短线拉升较急）
    > 6.0%  — 短线急涨过陡，回踩力度可能很大，不入选

  为什么 MA5 上限(6%)比 MA10 上限(4%)宽松:
    MA5 对近期价格更敏感，一两根大阳线就能让 MA5 斜率飙升；
    而 MA10 更平滑，如果连 MA10 斜率都超过 4%，说明涨势
    确实过急。MA5 适当放宽到 6% 避免误杀正常波动。

辅助验证:
  MA10 方向向上: MA10_T > MA10_(T-1)
  MA20 方向向上: MA20_T > MA20_(T-1)

  MA5 + MA10 + MA20 全部向上为"强多头"，至少 MA10 向上才可考虑。
  （MA5 在回踩阴线当日可能微幅走平或拐头，不作为硬性过滤条件，
   而是在评分中体现。）
```

#### 条件 2：均线粘合度 — MA10 距离 MA20、MA30 不太远

均线过度发散意味着短线涨幅已大，回踩时容易跌穿多条均线。理想状态是各均线保持一定间距但不太远，说明趋势有序推进而非急拉。

```
粘合度指标:
  spread_10_20 = (MA10_T - MA20_T) / MA20_T × 100%
  spread_10_30 = (MA10_T - MA30_T) / MA30_T × 100%

粘合条件:
  spread_10_20 <= 5.0%    （MA10 到 MA20 距离不超过 5%）
  spread_10_30 <= 8.0%    （MA10 到 MA30 距离不超过 8%）

评分细分:
  spread_10_20:
    <= 2%  — 紧密粘合（最佳，均线刚刚张开）
    2%~3.5% — 适度间距（良好）
    3.5%~5% — 偏宽（可接受，需叠加其他条件）
    > 5%   — 发散过大，不入选

  spread_10_30:
    <= 4%  — 紧密粘合（最佳）
    4%~6%  — 适度间距（良好）
    6%~8%  — 偏宽（需注意）
    > 8%   — 不入选
```

**为什么用 MA10 作为粘合度的锚点：** MA10 是本策略的核心支撑位（条件 3 要求价格回踩 MA10），因此衡量 MA10 与更长周期均线的距离最有意义——如果 MA10 离 MA20/MA30 太远，说明一旦 MA10 失守，下方有较大的空间。

#### 条件 3：当日是缩量阴线，且价格回踩至 MA10 附近

这是最核心的触发条件。在多头趋势中，缩量阴线回踩 MA10 是经典的洗盘形态。

```
缩量条件:
  vol_ratio = 当日成交量 / MA(成交量, 5)
  vol_ratio < 0.8

  更严格版本:
    vol_ratio < 0.6 → 显著缩量（加分）
    vol_ratio < 0.4 → 极度缩量（加分更多）

阴线条件:
  收盘价 < 开盘价（实体为阴）

  阴线实体不能过大（过大说明抛压重，不是温和洗盘）:
    阴线实体幅度 = (开盘 - 收盘) / 收盘
    要求: 阴线实体幅度 < 3%

    含义:
      < 1%  — 微幅小阴线（最佳洗盘形态）
      1%~2% — 小阴线（正常回踩）
      2%~3% — 中阴线（可接受）
      > 3%  — 较大阴线，可能不是简单洗盘

回踩 MA10 条件:
  distance_to_ma10 = (收盘价 - MA10_T) / MA10_T × 100%

  -2% <= distance_to_ma10 <= 2%

  含义:
    -2%~0%  — 略低于 MA10（最佳，刺穿后有望回升）
    0%~1%   — 贴近 MA10 上方（良好）
    1%~2%   — 略高于 MA10（可接受）
    低于-2% — 跌穿 MA10 过深，支撑可能已失效
    高于 2% — 还没到支撑位，回踩不充分

  额外约束: 价格离MA10比离MA5更近
    abs(收盘价 - MA10) <= abs(收盘价 - MA5)

    目的: 确保价格确实回踩到了MA10附近，而不是仅在MA5附近
          小幅波动就被误判为"回踩MA10"。如果价格离MA5更近，
          说明回踩不充分，还没下探到MA10的支撑区域。
```

#### 条件 4：过去一段时间有过放量上涨

没有前期放量上涨，就没有「主力介入」的证据。本条件确保这只股票确实有资金关注、有上攻历史，当前回踩才有「蓄力再攻」的逻辑基础。

```
回看窗口: 过去 20 个交易日内

放量上涨日定义:
  当日涨幅 >= 3%
  当日成交量 >= MA(成交量, 20) × 1.5

统计指标:
  surge_days = 满足放量上涨定义的天数

入选条件: surge_days >= 1

评分细分:
  surge_days >= 3  — 多次放量上涨，资金高度关注（最佳）
  surge_days == 2  — 有两次放量上涨（良好）
  surge_days == 1  — 有一次放量上涨（基础达标）
  surge_days == 0  — 近期无放量上涨，不入选

补充维度: 最大单日放量涨幅
  max_surge_pct = max(放量上涨日的涨幅)

  max_surge_pct >= 7%  — 有过涨停级别拉升（强关注）
  max_surge_pct 5%~7%  — 有过大阳线（良好）
  max_surge_pct 3%~5%  — 有过放量阳线（基础达标）
```

### 2.2 补充过滤条件

| 条件 | 规则 | 原因 |
|------|------|------|
| 排除 ST | 名称不含 ST / *ST | 退市风险 |
| 排除北交所 | 代码不以 8/4 开头 | 流动性和涨跌幅机制不同 |
| 排除科创、创业板 | 代码不以 68/3 开头 | 交易门槛限制 |
| 流通市值 | 30 亿 ~ 500 亿 | 排除极端小盘和超大盘 |
| 涨幅位置 | 距 60 日最低价涨幅 < 40% | 排除已涨太多的标的 |
| 数据量 | 至少 60 个交易日数据 | 保证均线和放量统计可靠 |

### 2.3 选股条件总结

```
信号触发 = {
  全市场股票
  ∩ 均线多头排列 (MA5 > MA10 > MA20 > MA30)
  ∩ 均线斜率向上且温和 (0.5% <= slope_ma10 <= 4.0%, slope_ma5 <= 6.0%)
  ∩ 均线粘合 (spread_10_20 <= 5%, spread_10_30 <= 8%)
  ∩ 当日缩量阴线 (vol_ratio < 0.8, 阴线, 实体 < 3%)
  ∩ 价格回踩MA10 (-2% <= distance_to_ma10 <= 2%)
  ∩ 过去20日有放量上涨 (surge_days >= 1)
  ∩ 非 ST / 非北交所 / 非科创 / 非创业板
  ∩ 流通市值 30~500亿
  ∩ 涨幅位置 < 40%
}
```

## 三、量化评分模型

### 3.1 评分维度与权重

| 维度 | 权重 | 评分范围 | 说明 |
|------|------|---------|------|
| A. 均线质量 | 25% | 0-100 | 排列健康度、斜率方向与幅度 |
| B. 均线粘合度 | 20% | 0-100 | MA10 与 MA20/MA30 的距离 |
| C. 回踩质量 | 25% | 0-100 | 缩量程度、阴线实体、距 MA10 距离 |
| D. 放量历史 | 20% | 0-100 | 放量上涨次数、最大涨幅 |
| E. 环境加分 | 10% | 0-100 | 市场温度 |

### 3.2 各维度评分细则

#### A. 均线质量（25%）

```
均线向上方向数:
  up_count = 0
  if MA5_T  > MA5_(T-1):  up_count += 1
  if MA10_T > MA10_(T-1): up_count += 1
  if MA20_T > MA20_(T-1): up_count += 1
  if MA30_T > MA30_(T-1): up_count += 1

  up_count == 4: base = 50   （全部向上，强多头）
  up_count == 3: base = 35   （三线向上）
  up_count == 2: base = 20   （两线向上，最低要求）

MA10 斜率分:
  slope = (MA10_T - MA10_(T-5)) / MA10_(T-5) × 100%

  0.5%~1.0%:  +30  （温和上行，最理想）
  1.0%~2.0%:  +25  （稳健上行）
  2.0%~3.0%:  +15  （偏快）
  3.0%~4.0%:  +5   （偏陡，扣分）

MA5 斜率分（方向 + 陡峭度双重评估）:
  slope_ma5 = (MA5_T - MA5_(T-3)) / MA5_(T-3) × 100%

  方向分（回踩阴线当日 MA5 可能微拐头，区别对待）:
    slope_ma5 > 0:           +5  （MA5 仍向上，强）
    -0.5% < slope_ma5 <= 0:  +3  （MA5 微幅走平，可接受）
    slope_ma5 <= -0.5%:       +0  （MA5 明显拐头，减弱）

  陡峭度分（过陡说明短线急涨，回踩风险大）:
    slope_ma5 <= 2.0%:        +10  （温和，最理想）
    2.0% < slope_ma5 <= 4.0%: +5   （中等）
    4.0% < slope_ma5 <= 6.0%: +0   （偏陡，不加分）

最终 A 分 = min(base + MA10斜率分 + MA5方向分 + MA5陡峭度分, 100)
```

#### B. 均线粘合度（20%）

```
spread_10_20 = (MA10 - MA20) / MA20 × 100%
spread_10_30 = (MA10 - MA30) / MA30 × 100%

MA10-MA20 粘合分:
  <= 1%:    45
  1%~2%:    40
  2%~3%:    30
  3%~4%:    20
  4%~5%:    10

MA10-MA30 粘合分:
  <= 2%:    45
  2%~4%:    40
  4%~6%:    30
  6%~8%:    15

加分项: 三线粘合（MA20 到 MA30 距离也小）
  spread_20_30 = (MA20 - MA30) / MA30 × 100%
  spread_20_30 <= 2%: +10（三线紧密，蓄势充分）

最终 B 分 = min(粘合分之和, 100)
```

#### C. 回踩质量（25%）

这是最关键的维度，评估当天缩量阴线回踩的精准度。

```
缩量分:
  vol_ratio = 当日量 / MA5量
  < 0.4:   40  （极度缩量，抛压极轻）
  0.4~0.6: 30  （显著缩量）
  0.6~0.8: 20  （适度缩量）

阴线实体分（实体越小说明抛压越轻）:
  body_pct = (开盘 - 收盘) / 收盘 × 100%
  < 0.5%:  20  （十字星级别）
  0.5%~1%: 15  （小阴线）
  1%~2%:   10  （中小阴线）
  2%~3%:   5   （中阴线）

距 MA10 距离分（越贴近越好）:
  dist = abs(收盘 - MA10) / MA10 × 100%
  < 0.5%:  30  （精准触及）
  0.5%~1%: 25  （非常接近）
  1%~1.5%: 15  （适度距离）
  1.5%~2%: 5   （偏远）

加分项: 下影线支撑
  如果最低价 < MA10 但收盘价 >= MA10 × 0.99:
    +10  （盘中跌破 MA10 后拉回，支撑有效）

最终 C 分 = min(缩量分 + 实体分 + 距离分 + 加分, 100)
```

#### D. 放量历史（20%）

```
surge_days（过去 20 日放量上涨天数）:
  >= 3:  50
  == 2:  40
  == 1:  25

max_surge_pct（最大单日放量涨幅）:
  >= 7%: 30  （涨停级别）
  5%~7%: 25
  3%~5%: 15

放量上涨的时效性（最近一次放量上涨距今天数）:
  <= 5日:    +20  （刚刚放量过，资金记忆新鲜）
  6~10日:    +15
  11~15日:   +10
  16~20日:   +5   （较久远，信号衰减）

最终 D 分 = min(天数分 + 涨幅分 + 时效分, 100)
```

#### E. 环境加分（10%）

```
直接映射 market_temperature:
  score >= 50: 环境分 = score
  score < 50:  环境分 = score × 0.6（弱势市场打折）
```

### 3.3 总分与操作建议

```
总分 = A×0.25 + B×0.20 + C×0.25 + D×0.20 + E×0.10
```

| 总分 | 级别 | 操作建议 |
|------|------|---------|
| >= 75 | **S — 极佳** | 半仓买入，经典的多头回踩形态 |
| 60-74 | **A — 良好** | 1/3 仓买入，止损设在 MA20 下方 |
| 45-59 | **B — 可关注** | 加入观察池，次日若确认反弹可轻仓试探 |
| < 45 | **C — 较弱** | 忽略，信号质量不高 |

### 3.4 硬性否决规则

无论总分多高，出现以下情况直接排除：

1. **MA20 方向向下**：中期趋势已拐头，多头排列可能是假象
2. **近 5 日有跌停**：可能是暴雷而非正常回踩
3. **当日阴线实体 > 5%**：大阴线不是洗盘，是恐慌抛售
4. **市场处于冰点期**（market_temperature < 20）：系统性风险
5. **均线排列持续不足 3 天**：多头排列刚形成就回踩，不稳定

## 四、操作流程

```
Step 1  收盘后扫描 (5 分钟)
        ├─ 全市场筛选: 四大核心条件
        ├─ 补充过滤: 排除ST/北交所/高位等
        └─ 输出评分排序的候选列表

Step 2  评估环境 (2 分钟)
        ├─ 检查 market_temperature 分数
        ├─ 冰点期 → 全部放弃
        └─ 市场温度 >= 40 → 正常操作

Step 3  确定操作 (3 分钟)
        ├─ S级(>=75): 次日开盘买入（半仓）
        ├─ A级(60-74): 次日开盘买入（1/3仓）
        ├─ B级(45-59): 观察次日走势
        │     └─ 次日低开缩量 → 再次确认，可轻仓
        │     └─ 次日反弹放量 → 可追入
        └─ C级(<45): 不操作

Step 4  买入执行
        ├─ 次日集合竞价或开盘 15 分钟内
        ├─ 挂单价: 前一日收盘价附近
        └─ 如果次日跳空高开 > 2% → 放弃追高，等下一次回踩
```

## 五、止损与止盈

### 5.1 止损规则

| 规则 | 触发条件 | 执行 |
|------|---------|------|
| 硬止损 | 收盘价 < 买入价 × (1 - 5%) | 次日开盘卖出 |
| MA20 止损 | 收盘价连续 2 日 < MA20 | 第 3 日开盘卖出 |
| 均线破位 | MA5 下穿 MA10 且 MA10 拐头向下 | 次日开盘卖出 |

为什么硬止损是 5% 而不是 8%：本策略买入点贴近 MA10，如果回踩有效，跌破 5% 大概率说明 MA10 支撑已失效。与趋势策略（突破买入、止损 8%）不同，回踩买入的止损应更紧。

### 5.2 止盈规则

| 规则 | 触发条件 | 执行 |
|------|---------|------|
| 放量大阳止盈 | 持仓期间出现涨幅 > 5% 的放量大阳线 | 卖出 50%，剩余移动止盈 |
| 移动止盈 | 从最高点回撤 5% | 全部卖出 |
| 时间止损 | 买入后 5 个交易日未上涨 3% | 减仓至 50%，等待或止损 |

### 5.3 优先级

```
硬止损(5%) > MA20止损 > 均线破位 > 移动止盈 > 时间止损
```

### 6.1 理想形态 — 温和多头回踩小阴线

```
日期    K线    成交量      均线状态

D-10   阳线    ████       放量上涨（条件4触发）
D-9    阳线    ███        温和上涨
D-8    阳线    ██         缩量上涨
D-7    阴线    █          缩量回调
D-6    阳线    ██         反弹
D-5    阳线    ████       再次放量上涨（条件4再次触发）
D-4    阳线    ███        上涨
D-3    十字星  █          缩量犹豫
D-2    阴线    █          缩量回落
D-1    阴线    █          缩量回落

D      小阴线  ▌          ← 极度缩量小阴线
       价格: 接近MA10      ← 回踩至MA10
       均线: MA5>10>20>30  ← 多头排列
       MA10-MA20距离: 2%   ← 粘合良好

       → ★★★★ 信号触发！S级买入机会
```

### 6.2 警示形态 — 不应买入的情况

```
场景1: 大阴线回踩（阴线实体 > 3%）
  → 不是温和洗盘，可能是主力出货，排除

场景2: 均线过度发散（MA10 距 MA20 超过 5%）
  → 短线涨幅已大，即使回踩MA10也可能继续下探至MA20，排除

场景3: 放量阴线回踩（量比 > 1）
  → 放量下跌说明有大单在卖，不是洗盘而是出货，排除

场景4: 无前期放量上涨
  → 没有资金进场的证据，多头排列可能是缓慢爬升而非趋势启动，排除
```

## 十、伪代码

### 10.1 核心检测函数

```python
def check_pullback_ma10(df, target_date):
    """
    检测指定日期是否满足多头回踩MA10缩量阴线条件。

    参数:
        df: 日K数据 DataFrame，包含 '收盘','开盘','最高','最低','成交量','涨跌幅'
        target_date: 目标日期

    返回:
        PullbackMA10Result or None
    """
    # 计算均线
    for p in [5, 10, 20, 30]:
        df[f'MA{p}'] = df['收盘'].rolling(p).mean()

    t = get_row(df, target_date)
    t_prev = get_prev_row(df, target_date)

    # ── 条件1: 均线多头排列 + 斜率温和 ──
    if not (t.MA5 > t.MA10 > t.MA20 > t.MA30):
        return None

    slope_ma10 = (t.MA10 - get_ma10_5days_ago(df)) / get_ma10_5days_ago(df) * 100
    if not (0.5 <= slope_ma10 <= 4.0):
        return None

    slope_ma5 = (t.MA5 - get_ma5_3days_ago(df)) / get_ma5_3days_ago(df) * 100
    if slope_ma5 > 6.0:
        return None

    # ── 条件2: 均线粘合度 ──
    spread_10_20 = (t.MA10 - t.MA20) / t.MA20 * 100
    spread_10_30 = (t.MA10 - t.MA30) / t.MA30 * 100
    if spread_10_20 > 5.0 or spread_10_30 > 8.0:
        return None

    # ── 条件3: 缩量阴线 + 回踩MA10 ──
    vol_ratio = t.volume / ma5_volume
    is_yin = t.close < t.open
    body_pct = (t.open - t.close) / t.close * 100
    dist_to_ma10 = (t.close - t.MA10) / t.MA10 * 100

    if not (vol_ratio < 0.8 and is_yin and body_pct < 3.0):
        return None
    if not (-2.0 <= dist_to_ma10 <= 2.0):
        return None

    # ── 条件4: 过去20日有放量上涨 ──
    surge_days = count_surge_days(df, target_date, lookback=20)
    if surge_days < 1:
        return None

    # ── 评分 ──
    score_a = calc_ma_quality_score(...)
    score_b = calc_cohesion_score(spread_10_20, spread_10_30)
    score_c = calc_pullback_score(vol_ratio, body_pct, dist_to_ma10, ...)
    score_d = calc_surge_history_score(surge_days, max_surge_pct, days_since)
    score_e = calc_env_score(market_temperature)

    total = score_a*0.25 + score_b*0.20 + score_c*0.25 + score_d*0.20 + score_e*0.10

    return PullbackMA10Result(
        total_score=total,
        grade=grade_from_score(total),
        ...
    )
```

### 10.2 放量上涨日统计

```python
def count_surge_days(df, target_date, lookback=20):
    """
    统计过去 lookback 个交易日内的放量上涨天数。

    放量上涨定义:
      涨幅 >= 3% 且 成交量 >= MA20量 × 1.5
    """
    window = df.loc[target_idx - lookback : target_idx - 1]
    ma20_vol = df['成交量'].rolling(20).mean()

    surge_mask = (
        (window['涨跌幅'] >= 3.0)
        & (window['成交量'] >= ma20_vol * 1.5)
    )

    surge_days = surge_mask.sum()
    if surge_days > 0:
        max_surge_pct = window.loc[surge_mask, '涨跌幅'].max()
        last_surge_idx = window.loc[surge_mask].index[-1]
        days_since = target_idx - last_surge_idx
    else:
        max_surge_pct = 0
        days_since = 999

    return surge_days, max_surge_pct, days_since
```

## 十一、策略局限与注意事项

### 11.1 策略固有局限

| 局限 | 说明 | 应对 |
|------|------|------|
| 熊市假多头 | 下跌趋势中偶尔出现短暂的多头排列 | 结合 market_temperature，冰点期不操作 |
| MA10 不一定撑住 | 回踩 MA10 后可能继续下探 MA20 | 设好止损（5%），不恋战 |
| 慢牛错过 | 有些慢牛股回踩幅度小，不到 MA10 就拉起来 | 这种情况宁可错过，不降低标准 |
| 急涨后的回踩 | 斜率过大时回踩力度也大 | 斜率上限控制在 4%，排除急涨标的 |
| 盘整中的假信号 | 价格在均线附近反复纠缠 | 要求均线排列至少持续 3 天，过滤刚排列就回踩的情况 |

### 11.2 最佳适用环境

| 市场环境 | 信号可信度 | 建议 |
|----------|-----------|------|
| 牛市/主升期 | 高 | 正常使用，回踩大概率有效 |
| 震荡市 | 中 | 仅关注 S/A 级信号，快进快出 |
| 熊市/下跌期 | 低 | 不使用，均线多头排列极少且不稳定 |

### 11.3 与均线回归策略的区别

| 对比项 | 均线回归策略 (ma_alignment) | 回踩MA10策略（本策略） |
|--------|--------------------------|---------------------|
| 买入时机 | 多头排列的第一天 | 多头排列确立后的回踩阴线 |
| 核心逻辑 | 趋势刚确立，从无到有的拐点 | 趋势已确立，等待回调做多 |
| 量价关系 | 不强调量能 | 要求当日缩量 + 历史有放量上涨 |
| 均线要求 | 首日排列即可 | 排列需持续一段时间，且保持粘合 |
| 风险特征 | 可能是假回归（排列后又乱） | 可能回踩不止（跌穿MA10） |
| 互补关系 | 先发现趋势确立 → 跟踪持续 → 若回踩则切换到本策略 |

两个策略可以组合使用：均线回归策略发现趋势启动信号 → 跟踪观察 → 当出现回踩 MA10 缩量阴线时，由本策略给出买入信号。

### 11.4 与缩量主升浪策略的区别

| 对比项 | 缩量主升浪 (trend_surge) | 回踩MA10策略（本策略） |
|--------|--------------------------|---------------------|
| 入场逻辑 | 缩量横盘后的放量突破 | 多头趋势中的缩量回踩 |
| 买入信号 | 放量 + 突破 + 阳线 | 缩量 + 回踩 + 阴线 |
| 趋势阶段 | 趋势启动初期 | 趋势运行中期 |
| 心态要求 | 在突破时果断追入 | 在阴线时逆向买入 |

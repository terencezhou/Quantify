# 缩量下跌触及支撑位 — 买入信号分析框架

## 一、核心命题

> **缩量下跌触及支撑位，是上涨信号吗？**

**结论：不一定。它是一个"值得关注"的信号，但需要多维度确认后才能成为"值得操作"的信号。**

缩量下跌触及支撑位的本质含义是：
- **缩量** → 卖方力量衰竭，抛压减轻
- **下跌** → 短期趋势仍然向下
- **触及支撑位** → 到达历史上多头曾经反击成功的价格区域

三者叠加说明：**空方已无力继续打压，价格落入多方有意愿防守的区域**。这是潜在反转的必要条件，但不是充分条件。

---

## 二、信号拆解 — 三个维度逐一验证

### 维度 1：缩量的质量

并非所有缩量都有意义，需要区分"真缩量"和"假缩量"。

| 判断项 | 有效缩量 | 无效缩量 |
|--------|----------|----------|
| 量比关系 | 当日成交量 < 5日均量的 60% | 量只是略低于均量（80%~100%） |
| 缩量持续性 | 连续 2~3 日递减 | 单日缩量，次日又放量下跌 |
| 缩量阶段 | 下跌中后段（已跌 3 日以上） | 下跌初期（高位刚开始回调 1~2 日） |
| 换手率 | < 3%（主力未大规模出逃） | 换手率仍 > 5%（仍有大量筹码在交换） |

**关键指标**：
```
volume_ratio = 当日成交量 / MA(成交量, 5)
有效缩量条件: volume_ratio < 0.6 且 连续缩量天数 >= 2
```

### 维度 2：支撑位的强度

支撑位不是一条精确的线，而是一个区域。支撑的强度取决于其"历史记忆"的深刻程度。

**支撑位的来源（按强度排序）**：

| 优先级 | 支撑类型 | 强度 | 判定方法 |
|--------|----------|------|----------|
| 1 | 前期密集成交区（筹码峰） | ★★★★★ | 筹码分布中的峰值区间，套牢盘集中 |
| 2 | 多次回踩不破的低点 | ★★★★ | 近 60 日内 >= 2 次触及该价位后反弹 |
| 3 | 关键均线（MA20/MA60/MA120） | ★★★ | 股价触及均线且均线仍向上或走平 |
| 4 | 整数关口 / 前期跳空缺口上沿 | ★★ | 心理支撑，精确度较低 |
| 5 | 斐波那契回撤位（38.2%/50%/61.8%） | ★★ | 从前一轮上涨的起点到高点计算 |

**关键判定**：
```
支撑位有效条件:
  1. 股价进入支撑区域（支撑位 ± 2% 范围内）
  2. 该支撑位此前至少被验证过 1 次（触及后反弹 >= 3%）
  3. 支撑位类型优先级 <= 3（筹码峰 / 多次低点 / 关键均线）
```

### 维度 3：下跌的性质

需要判断这轮下跌是"回调"还是"反转"。

| 特征 | 健康回调（可买） | 趋势反转（不可买） |
|------|------------------|---------------------|
| 中期趋势 | MA20/MA60 仍向上或走平 | MA20 已拐头向下，MA60 走平或下行 |
| 跌幅 | 从近期高点回撤 5%~15% | 从高点已跌 > 20% |
| 下跌速度 | 阴线实体小，缓跌 | 大阴线频出，连续跌停 |
| MACD 状态 | 快线在零轴上方回落 | 快线已下穿零轴 |
| 板块联动 | 同板块个股多数企稳 | 同板块全线杀跌 |

## 三、确认信号 — 等待"第二根 K 线"

**核心原则：信号出现当天不操作，等待次日确认。**

缩量触支撑是"预备信号"，需要后续 K 线给出确认：

### 强确认（可以介入）

| 确认形态 | 描述 | 可信度 |
|----------|------|--------|
| 放量阳线 | 次日低开高走，收阳线，量比 > 1.5 | ★★★★★ |
| 缩量十字星 + 次日阳线 | 当日极度缩量收十字星，后一日放量收阳 | ★★★★ |
| 下影线探底 | 当日下探支撑位后拉回，留长下影线（下影 > 实体 2 倍） | ★★★★ |
| 尾盘异动 | 分时图尾盘 30 分钟出现资金流入脉冲 | ★★★ |

### 弱确认 / 需继续观察

| 形态 | 描述 | 处理 |
|------|------|------|
| 继续缩量收小阴线 | 未破支撑但也没反弹 | 继续等待，不急于操作 |
| 平开平收 | 窄幅震荡，犹豫不决 | 观察资金流方向 |

### 失败信号（放弃）

| 形态 | 描述 | 处理 |
|------|------|------|
| 放量破位 | 次日放量跌破支撑位 | 支撑失效，立即放弃 |
| 连续缩量阴跌不止 | 缩量但持续 5 日以上阴跌 | 不是洗盘，而是阴跌出货 |
| 跳空低开 | 次日大幅低开（> 2%）且不回补 | 主力不护盘，趋势恶化 |

## 四、环境过滤 — 宏观与中观条件

即使个股信号完美，如果大环境不配合，成功率也会大幅下降。

### 4.1 市场环境（依赖 market_temperature）

| 市场阶段 | 信号可信度 | 建议 |
|----------|-----------|------|
| 冰点期（score < 20） | 低 | 不操作，支撑位大面积失效 |
| 混沌/修复（20~50） | 中 | 仅限龙头股或强势股 |
| 主升/发酵（50~80） | 高 | 正常操作，信号可信 |
| 高潮期（> 80） | 中 | 注意可能是诱多反弹后继续下杀 |

### 4.2 板块热度（依赖 industry_temperature）

| 板块状态 | 影响 |
|----------|------|
| 板块处于启动/发酵期 | 加分：板块资金仍在流入，个股回调是上车机会 |
| 板块处于高潮/退潮期 | 减分：个股回调可能是板块退潮的开始 |
| 板块无明显热度 | 中性：纯粹看个股自身技术面 |

### 4.3 龙头状态（依赖 leader_stock）

| 条件 | 判断 |
|------|------|
| 该股本身是板块龙头 | 龙头回调触支撑，可信度最高 |
| 板块龙头仍然强势 | 跟风股回调触支撑，仍有反弹空间 |
| 板块龙头已经走弱/补跌 | 跟风股支撑大概率不保 |

## 五、量化打分模型

将上述分析量化为一个 0~100 的信号评分。

### 5.1 评分维度与权重

| 维度 | 权重 | 评分范围 | 说明 |
|------|------|----------|------|
| A. 缩量质量 | 20% | 0~100 | volume_ratio、连续缩量天数、换手率 |
| B. 支撑强度 | 25% | 0~100 | 支撑类型、历史验证次数、偏离度 |
| C. 下跌性质 | 20% | 0~100 | 中期趋势、回撤幅度、MACD 位置 |
| D. 确认形态 | 15% | 0~100 | 次日K线形态、分时资金流 |
| E. 市场环境 | 10% | 0~100 | market_temperature 得分映射 |
| F. 板块/龙头 | 10% | 0~100 | 板块热度 + 龙头状态 |

### 5.2 各维度评分细则

#### A. 缩量质量（20%）

```
base = 0
if volume_ratio < 0.4:  base = 80
elif volume_ratio < 0.6: base = 60
elif volume_ratio < 0.8: base = 30
else: base = 0

连续缩量加分: +10/天 (最多+20)
换手率 < 2%: +10
换手率 > 5%: -20（大量换手说明分歧大）
```

#### B. 支撑强度（25%）

```
按支撑类型基础分:
  筹码峰:    80
  多次低点:   70
  关键均线:   60
  缺口/整数:  40
  斐波那契:   40

历史验证次数加分: +10/次 (最多+20)
当前价距支撑位偏离: <= 1% 满分, 1%~3% 按比例, > 3% 不算触及
```

#### C. 下跌性质（20%）

```
MA20 方向:  向上+30, 走平+15, 向下+0
回撤幅度:   5%~10% → 80,  10%~15% → 60,  15%~20% → 30,  >20% → 0
MACD:      快线在零轴上方 +20, 零轴附近 +10, 零轴下方 +0
```

#### D. 确认形态（15%）

```
放量阳线:     90
长下影线:     75
缩量十字星:   50（需等次日再确认）
尾盘资金流入: 60
无明显确认:   20
放量破位:     0（一票否决）
```

#### E. 市场环境（10%）

```
直接映射 market_temperature score:
  score >= 50: 环境分 = score
  score < 50:  环境分 = score * 0.5（弱势市场打折）
```

#### F. 板块/龙头（10%）

```
板块启动/发酵 + 龙头强势: 90
板块启动/发酵 + 龙头走平: 60
板块高潮/退潮:           30
无明显板块归属:           50
该股本身是龙头:           额外 +20（封顶100）
```

### 5.3 总分与操作建议

```
总分 = A*0.20 + B*0.25 + C*0.20 + D*0.15 + E*0.10 + F*0.10
```

| 总分区间 | 信号等级 | 操作建议 |
|----------|----------|----------|
| >= 75 | 强买入信号 | 可建仓（半仓），次日确认后加仓 |
| 60~74 | 中等信号 | 小仓试探（1/4 仓），严格止损 |
| 45~59 | 弱信号 | 仅加入观察池，不操作 |
| < 45 | 无效信号 | 忽略 |

### 5.4 硬性否决规则

无论总分多高，出现以下情况直接判为无效：

1. **放量跌破支撑位**（当日或次日）：支撑失效
2. **市场处于冰点期**（market_temperature < 20）：系统性风险
3. **该股近 5 日有跌停**：可能是暴雷，不是正常回调
4. **连续阴跌超过 7 日**：阴跌出货形态，非洗盘

## 六、操作流程

```
Step 1  扫描信号
        ├─ 全市场筛选: 当日缩量(volume_ratio < 0.6) 且 触及支撑位
        └─ 输出候选列表

Step 2  环境过滤
        ├─ 市场冰点? → 全部放弃
        └─ 逐只检查板块热度与龙头状态

Step 3  逐只评分
        ├─ 计算 A~F 六维度得分
        ├─ 检查硬性否决规则
        └─ 输出带分数的候选列表（>= 60 分入选）

Step 4  等待确认（次日）
        ├─ 次日开盘后观察 K 线形态
        ├─ 更新 D 维度（确认形态）得分
        └─ 重新计算总分

Step 5  执行操作
        ├─ 总分 >= 75: 建仓（半仓），止损设在支撑位下方 3%
        ├─ 总分 60~74: 小仓试探（1/4 仓），止损同上
        └─ 总分 < 60: 放弃或继续观察
```

## 七、与现有模块的依赖关系

```
DataManager
  ├─ 日K线数据    → 计算缩量、均线、MACD、回撤幅度
  ├─ 分时图数据   → 尾盘资金脉冲检测
  ├─ 资金流数据   → 主力净流入/流出趋势
  └─ 筹码分布     → 支撑位（筹码峰）识别

MarketTemperature  → E 维度: 市场环境分
IndustryTemperature → F 维度: 板块热度
LeaderStockFinder   → F 维度: 龙头状态
```

## 八、常见误判场景

| 场景 | 表现 | 为什么会失败 |
|------|------|-------------|
| 阴跌缩量 | 每天缩一点量，跌一点价 | 不是卖方衰竭，而是买方完全消失 |
| 假突破后回落 | 刚放量突破又缩量回落到支撑 | 前期放量是出货，缩量回调是续跌 |
| 利空未出尽 | 技术面完美但基本面有隐患 | 业绩雷、监管风险等不反映在技术指标中 |
| 大盘系统性下跌 | 个股信号好但大盘崩了 | 覆巢之下无完卵 |

**规避方法**：硬性否决规则 + 市场环境过滤 + 仓位控制 + 严格止损。
